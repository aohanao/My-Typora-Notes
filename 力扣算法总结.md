## 1 哈希

#### 1.两数之和

题目：给定一个整数数组 `nums` 和一个整数目标值 `target`，请你在该数组中找出 **和为目标值** *`target`* 的那 **两个** 整数，并返回它们的数组下标。你可以假设每种输入只会对应一个答案，并且你不能使用两次相同的元素。你可以按任意顺序返回答案。

思路：遍历数组时，用哈希表存储 `{值: 索引}`。对于当前数 `num`，检查 `target - num` 是否在表中，若存在则找到答案

```python
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        hashtable = dict()
        for i, num in enumerate(nums):
            if target - num in hashtable:
                return [hashtable[target - num], i]
            hashtable[nums[i]] = i 
        return []
```

时间复杂度：`O(n)`，其中 `n` 是数组中的元素数量

#### 49.字母异位词分组

题目：给你一个字符串数组，请你将 字母异位词 组合在一起。可以按任意顺序返回结果列表。

思路：异位词排序后结果相同（如 `"eat"` 和 `"tea"` 排序后都是 `"aet"`）。用排序后的字符串作为哈希表的 key，将原字符串分组

```python
class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        mp = collections.defaultdict(list)
        for st in strs:
            key = "".join(sorted(st))
            mp[key].append(st)
        return list(mp.values())
```

时间复杂度：`O(nklogk)`，其中 `n` 是 `strs` 中的字符串的数量，`k` 是 `strs` 中的字符串的的最大长度。需要遍历 `n` 个字符串，对于每个字符串，需要 `O(klogk)` 的时间进行排序以及 `O(1)` 的时间更新哈希表，因此总时间复杂度是 `O(nklogk)`

#### 128.最长连续序列

题目：给定一个未排序的整数数组 `nums` ，找出数字连续的最长序列（不要求序列元素在原数组中连续）的长度。请你设计并实现时间复杂度为 `O(n)` 的算法解决此问题。

思路：将所有数存入集合，只从"连续序列起点"（即 `num-1` 不在集合中）开始向右遍历，计算序列长度

```python
class Solution:
    def longestConsecutive(self, nums: List[int]) -> int:
        longest_streak = 0
        num_set = set(nums)
        for num in num_set:
            if num - 1 not in num_set:
                current_num = num
                current_streak = 1
                while current_num + 1 in num_set:
                    current_num += 1
                    current_streak += 1
                longest_streak = max(longest_streak, current_streak)
        return longest_streak
```

时间复杂度：`O(n)`，其中 `n`为数组的长度

## 2 双指针

#### 283.移动零

题目：给定一个数组 `nums`，编写一个函数将所有 `0` 移动到数组的末尾，同时保持非零元素的相对顺序。**请注意** ，必须在不复制数组的情况下原地对数组进行操作。

思路：`left` 指向待填充位置，`right` 遍历数组。遇到非零数就交换到 `left` 位置，`left` 前进。保证 `[0, left)` 都是非零数

```python
class Solution:
    def moveZeroes(self, nums: List[int]) -> None:
        n = len(nums)
        left = right = 0
        while right < n:
            if nums[right] != 0:
                nums[left], nums[right] =  nums[right], nums[left]
                left += 1
            right += 1
```

时间复杂度：`O(n)`

#### 11.盛最多水的容器

题目：给定一个长度为 `n` 的整数数组 `height` 。有 `n` 条垂线，第 `i` 条线的两个端点是 `(i, 0)` 和 `(i, height[i])` 。找出其中的两条线，使得它们与 `x` 轴共同构成的容器可以容纳最多的水。返回容器可以储存的最大水量。**说明：**你不能倾斜容器。

思路：面积 = 底 × 高。底边长度固定时，移动较短的那根板（因为移动长的板，底边变小、高度不会增加，面积必然变小）

```python
class Solution:
    def maxArea(self, height: List[int]) -> int:
        n = len(height)
        left = 0
        right = n - 1
        ml = 0
        maxml = 0
        while left < right:
            ml = (right - left) * min(height[left], height[right])
            maxml =  max(maxml, ml)
            if height[left] < height[right]:
                left += 1
            else:
                right -= 1
        return maxml
```

时间复杂度：`O(n)`

#### 15.三数之和

题目：给你一个整数数组 `nums` ，判断是否存在三元组 `[nums[i], nums[j], nums[k]]` 满足 `i != j`、`i != k` 且 `j != k` ，同时还满足 `nums[i] + nums[j] + nums[k] == 0` 。请你返回所有和为 `0` 且不重复的三元组。**注意：**答案中不可以包含重复的三元组。

思路：先排序，固定第一个数，后两个数用双指针从两端向中间找。注意跳过重复元素避免重复解

```python
class Solution:
    def threeSum(self, nums: List[int]) -> List[List[int]]:
        n = len(nums)
        nums.sort()
        ans = list()
        for first in range(n):
            if first > 0 and nums[first] == nums[first - 1]:
                continue
            third = n - 1
            target = -nums[first]
            for second in range(first + 1, n):
                if second > first + 1 and nums[second] == nums[second - 1]:
                    continue
                while second < third and nums[second] + nums[third] > target:
                    third -= 1
                if second == third:
                    break
                if nums[second] + nums[third] == target:
                    ans.append([nums[first], nums[second], nums[third]])
        return ans
```

时间复杂度：`O(n^2)`

#### 43.接雨水

思路：哪边矮就先处理哪边（因为矮边决定水位）。用 `leftMax/rightMax` 记录两侧最大高度，当前位置能接的水 = `maxHeight - currentHeight`

```python
class Solution:
    def trap(self, height: List[int]) -> int:
        n = len(height)
        left, leftMax, rightMax = 0, 0, 0
        right = n - 1
        ans = 0
        while left < right:
            leftMax = max(leftMax, height[left])
            rightMax = max(rightMax, height[right])
            if height[left] < height[right]:
                ans += leftMax - height[left]
                left += 1
            else:
                ans += rightMax - height[right]
                right -= 1
        return ans
```

时间复杂度：`O(n)`，其中 `n` 是数组 `height` 的长度。两个指针的移动总次数不超过 `n`

## 3 滑动窗口

#### 3.无重复字符的最长子串

题目：给定一个字符串 `s` ，请你找出其中不含有重复字符的 **最长 子串** 的长度。

**子字符串** 是字符串中连续的 **非空** 字符序列

思路：右指针 `rk` 扩张窗口，用集合判断字符是否重复；遇到重复则左指针 `i` 右移收缩窗口（等价于发现重复 → 疯狂删左边直到不重复），记录窗口最大长度，

**左闭右开区间 [i, rk) 的长度 = rk - i ✅ 不用 +1**

**左闭右闭区间 [i, rk] 的长度 = rk - i + 1 ✅ 必须 +1**

```python
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        occ = set()
        n = len(s)
        rk, ans = 0, 0
        for i in range(n):
            if i != 0:
                occ.remove(s[i - 1])
            while rk < n and s[rk] not in occ:
                occ.add(s[rk])
                rk += 1
            ans = max(ans, rk - i)
        return ans
```

时间复杂度：`O(N)`，其中 *N* 是字符串的长度。左指针和右指针分别会遍历整个字符串一次。

#### 438.找到字符串中所有字母异位词

题目：给定两个字符串 `s` 和 `p`，找到 `s` 中所有 `p` 的 **异位词** 的子串，返回这些子串的起始索引。不考虑答案输出的顺序。

思路：窗口长度固定为 `p` 的长度，用哈希表统计字符频次。滑动时更新频次表，与 `p` 的频次表对比判断是否匹配

```python
from typing import List

class Solution:
    def findAnagrams(self, s: str, p: str) -> List[int]:
        s_len, p_len = len(s), len(p)
        if s_len < p_len:
            return []
        
        # 统计p的字符频次（字典）
        p_count = {}
        for char in p:
            p_count[char] = p_count.get(char, 0) + 1
        
        # 初始化窗口内s的字符频次（字典）
        s_count = {}
        for i in range(p_len):
            char = s[i]
            s_count[char] = s_count.get(char, 0) + 1
        
        ans = []
        # 初始窗口匹配检查
        if s_count == p_count:
            ans.append(0)
        
        # 滑动窗口更新
        for i in range(s_len - p_len):
            # 移出窗口左侧字符
            left_char = s[i]
            s_count[left_char] -= 1
            if s_count[left_char] == 0:  # 频次为0时删除键，避免干扰对比
                del s_count[left_char]
            
            # 移入窗口右侧新字符
            right_char = s[i + p_len]
            s_count[right_char] = s_count.get(right_char, 0) + 1
            
            # 对比当前窗口与p的频次
            if s_count == p_count:
                ans.append(i + 1)
        
        return ans
```

时间复杂度：O(m+(n−m)×Σ)，其中 n 为字符串 s 的长度，m 为字符串 p 的长度，Σ 为所有可能的字符数。我们需要 O(m) 来统计字符串 p 中每种字母的数量；需要 O(m) 来初始化滑动窗口；需要判断 n−m+1 个滑动窗口中每种字母的数量是否与字符串 p 中每种字母的数量相同，每次判断需要 O(Σ) 。因为 s 和 p 仅包含小写字母，所以 Σ=26。

## 4 子串

#### 560.和为k的子数组

题目：给你一个整数数组 `nums` 和一个整数 `k` ，请你统计并返回 *该数组中和为 `k` 的子数组的个数* 。子数组是数组中元素的连续非空序列。

思路：前缀和 + 哈希表，前缀和`s[i]` 表示 `nums[0..i]` 的和。若 `s[j] - s[i] = k`，则 `nums[i+1..j]` 和为 k。遍历时用哈希表记录前缀和出现次数，`ans += mp[s - k]`

```python
class Solution:
    def subarraySum(self, nums: List[int], k: int) -> int:
        mp = defaultdict(int) 
        s = ans = 0
        for num in nums:
            # 把「当前这个前缀和 s」出现的次数，加 1 次。mp 字典存的永远是：前缀和 = 出现了几次
            mp[s] += 1
            s += num
            ans += mp[s - k]
        return ans
```

时间复杂度：`O(n)`

#### 239.滑动窗口最大值

题目：给你一个整数数组 `nums`，有一个大小为 `k` 的滑动窗口从数组的最左侧移动到数组的最右侧。你只可以看到在滑动窗口内的 `k` 个数字。滑动窗口每次只向右移动一位。返回 *滑动窗口中的最大值* 。

思路：队列存的不是值，是下标，保证**下标对应的值从大到小**，**新来一个数**：把队列里比它小的**全部删掉**（留着没用），**删掉滑出窗口的过期数**（不在窗口里的不要），**窗口形成后**，**队首就是当前最大值**

```python
class Solution:
    def maxSlidingWindow(self, nums: List[int], k: int) -> List[int]:
        q = deque()
        ans = []
        for i, v in enumerate(nums):
            # 如果队列尾部元素 ≤ 当前元素，直接删掉
            while q and nums[q[-1]] <= v:
                q.pop()
            q.append(i)
            # 窗口范围：[i-k+1, i]
            if q[0] <= i-k:
                q.popleft()
            if i >= k - 1:
                ans.append(nums[q[0]])
        return ans
```

**时间复杂度**：**O(n)**

#### 76.最小覆盖子串

题目：给定两个字符串 `s` 和 `t`，长度分别是 `m` 和 `n`，返回 s 中的 **最短窗口 子串**，使得该子串包含 `t` 中的每一个字符（**包括重复字符**）。如果没有这样的子串，返回空字符串 `""`。

测试用例保证答案唯一。

思路：

```python
class Solution:
    def minWindow(self, s: str, t: str) -> str:
        left, right, valid, start, min_len = 0, 0, 0, 0, float('inf')
        need = Counter(t)
        n = collections.defaultdict(int)
        while right < len(s):
            c = s[right]
            right += 1
            if c in need:
                n[c] += 1
                if n[c] == need[c]:
                    valid += 1
            while valid == len(need):
                if right - left < min_len:
                    start = left
                    min_len = right - left
                d = s[left]
                left += 1
                if d in need:
                    if n[d] == need[d]:
                        valid -= 1
                    n[d] -= 1
        return "" if min_len == float('inf') else s[start : start + min_len]
```



## 5 普通数组

#### 53.最大子数组和

题目：给你一个整数数组 `nums` ，请你找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。**子数组**是数组中的一个连续部分。

思路：`current` 维护以当前元素结尾的最大子数组和。若 `current + x < x`，则丢弃前面的子数组，从 `x` 重新开始。`maxSum` 记录全局最大

```python
class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        # 初始化当前子数组和current、全局最大和maxSum为数组第一个元素
        # 因为子数组至少包含一个元素，初始值不能设为0（避免全负数的情况）
        current = maxSum = nums[0]
        # 从数组第二个元素开始遍历（第一个元素已作为初始值）
        for x in nums[1:]:
            # 情况1：将当前元素x加入前一个子数组，和为current + x
            # 情况2：以当前元素x为起点，重新开始一个子数组，和为x
            # 取两种情况的较大值作为新的当前子数组和
            current = max(x, current + x)
            # 更新全局最大和：比较当前子数组和与历史最大和，保留更大值
            maxSum = max(current, maxSum)
        return maxSum
```

#### 56.合并区间

题目：以数组 `intervals` 表示若干个区间的集合，其中单个区间为 `intervals[i] = [starti, endi]` 。请你合并所有重叠的区间，并返回 *一个不重叠的区间数组，该数组需恰好覆盖输入中的所有区间* 。

思路：按左端点升序，比较当前数组右端点与下一数组左端点，合并后存入新数组

```python
class Solution:
    def merge(self, intervals: List[List[int]]) -> List[List[int]]:
        # 先按照每个区间的起始位置进行排序
        intervals.sort(key=lambda x: x[0])
        merged = []  # 用于存放合并后的区间
        for interval in intervals:
            # 如果 merged 为空，或者当前区间和 merged 最后一个区间不重叠，则直接加入
            # 不重叠的判断条件：上一区间的结束 < 当前区间的开始
            if not merged or merged[-1][1] < interval[0]:
                merged.append(interval)
            else:
                # 否则，说明有重叠，合并区间：
                # 更新 merged 最后一个区间的结束位置为两者中较大的那个
                merged[-1][1] = max(merged[-1][1], interval[1])
        return merged
```

排序为 Python 内置的 `sort()` 方法，底层实现是 **Timsort** 算法，属于高效的比较类排序，时间复杂度固定为 **O(n log n)**（`n` 是区间数量）

#### 189.轮转数组

题目：给定一个整数数组 `nums`，将数组中的元素向右轮转 `k` 个位置，其中 `k` 是非负数。

思路：通过三次反转数组，实现数组向右旋转 k 位

```python
class Solution:
    def rotate(self, nums: List[int], k: int) -> None:
        n = len(nums)
        k %= n
        def reverse(start, end):
            while start < end:
                nums[start], nums[end] = nums[end], nums[start]
                start += 1
                end -= 1
        reverse(0, n - 1)
        reverse(0, k - 1)
        reverse(k, n - 1)
```

时间复杂度为: `O(n)`

#### 238.除了自身以外数组的乘积

题目：给你一个整数数组 `nums`，返回 数组 `answer` ，其中 `answer[i]` 等于 `nums` 中除了 `nums[i]` 之外其余各元素的乘积 。

题目数据 **保证** 数组 `nums`之中任意元素的全部前缀元素和后缀的乘积都在 **32 位** 整数范围内。

请 **不要使用除法，**且在 `O(n)` 时间复杂度内完成此题。

思路：双指针同步遍历数组，用两个变量分别累积左右乘积，一次遍历直接算出每个位置「左侧乘积 × 右侧乘积」的结果

```python
class Solution:
    def productExceptSelf(self, nums: List[int]) -> List[int]:
        n = len(nums)
        left = 0
        right = n - 1
        answer = [1] * n
        i = j = 1
        while left < n:
            answer[left] *= i
            i *= nums[left]
            answer[right] *= j
            j *= nums[right]
            left += 1
            right -= 1
        return answer
```

时间复杂度为: `O(n)`

#### 41.缺失的第一个正数

题目：给你一个未排序的整数数组 `nums` ，请你找出其中没有出现的最小的正整数。

请你实现时间复杂度为 `O(n)` 并且只使用常数级别额外空间的解决方案。

思路：我们要把 1 放到下标 0，2 放到下标 1，3 放到下标 2，数字 x 必须待在下标 x-1 的位置，最后遍历一遍，哪个位置不满足 `nums[i] == i+1`，i+1 就是答案

```python
class Solution:
    def firstMissingPositive(self, nums: List[int]) -> int:
        n = len(nums)
        for i in range(n):
            while 0 < nums[i] <= n and nums[nums[i] - 1] != nums[i]:
                target = nums[i] - 1
                nums[i], nums[target] = nums[target], nums[i]
        for i in range(n):
            if nums[i] != i + 1:
                return i + 1
        return n + 1
# 时间复杂度：O(N)，其中 N 是数组的长度。
# 空间复杂度：O(1)。
```



## 6 矩阵

#### 73.矩阵置零

题目：给定一个 `m x n` 的矩阵，如果一个元素为 **0** ，则将其所在行和列的所有元素都设为 **0** 。请使用 原地 算法。

思路：先用两个数组标记出**所有出现过 0 的行和列**，再根据这些标记，把这些行和列上的所有数字全部变成 0

```python
class Solution:
    def setZeroes(self, matrix: List[List[int]]) -> None:
        m = len(matrix)
        n = len(matrix[0])
        row = [False] * m
        col = [False] * n
        for i in range(m):
            for j in range(n):
                if matrix[i][j] == 0:
                    row[i] = col[j] = True
        for i in range(m):
            for j in range(n):
                if row[i] or col[j]:
                    matrix[i][j] = 0
```

时间复杂度：`O (m × n)`，m 是矩阵行数，n 是矩阵列数

#### 54.螺旋矩阵

题目：给你一个 `m` 行 `n` 列的矩阵 `matrix` ，请按照 **顺时针螺旋顺序** ，返回矩阵中的所有元素。

思路：用 `top, bottom, left, right` 四个变量，框住当前还没遍历的区域，走完一条边就收缩对应边界，方向就 `+1 % 4` 切换到下一个方向

```python
class Solution:
    def spiralOrder(self, matrix: List[List[int]]) -> List[int]:
        m, n = len(matrix), len(matrix[0])
        top, bottom = 0, m - 1
        left, right = 0, n - 1
        res = []
        direction = 0  # 0:右, 1:下, 2:左, 3:上
        while left <= right and top <= bottom:
            if direction == 0:  # 向右
                for col in range(left, right + 1):
                    res.append(matrix[top][col])
                top += 1
            elif direction == 1:  # 向下
                for row in range(top, bottom + 1):
                    res.append(matrix[row][right])
                right -= 1
            elif direction == 2:  # 向左
                for col in range(right, left - 1, -1):
                    res.append(matrix[bottom][col])
                bottom -= 1
            else:  # direction == 3，向上
                for row in range(bottom, top - 1, -1):
                    res.append(matrix[row][left])
                left += 1
            direction = (direction + 1) % 4  # 方向切换
        return res
```

时间复杂度：**O(m×n)**，每个元素只访问一次

空间复杂度：**O(1)**（除结果外无额外空间）

#### 48.旋转图像

题目：给定一个 *n* × *n* 的二维矩阵 `matrix` 表示一个图像。请你将图像顺时针旋转 90 度。

你必须在 原地 旋转图像，这意味着你需要直接修改输入的二维矩阵。**请不要** 使用另一个矩阵来旋转图像。

思路：顺时针旋转 90 度，矩阵中**任意一个元素**都会和另外**3 个元素**形成一个循环替换关系：`(i,j)` → `(j, n-i-1)` → `(n-i-1, n-j-1)` → `(n-j-1, i)` → `(i,j)`，只遍历矩阵 1/4 区域避免重复操作，坐标记法**新行 = 旧列 新列 = n - 1 - 旧行**

```python
class Solution:
    def rotate(self, matrix: List[List[int]]) -> None:
        n = len(matrix)
        # 只遍历上半部分行
        for i in range(n // 2):
            # 只遍历左半部分列
            for j in range((n + 1) // 2):
                matrix[i][j], \
                matrix[j][n - i - 1], \
                matrix[n - i - 1][n - j - 1], \
                matrix[n - j - 1][i] \
                = matrix[n - j - 1][i], \
                matrix[i][j], \
                matrix[j][n - i - 1], \
                matrix[n - i - 1][n - j - 1]
```

#### 240.搜索二维矩阵II

题目：编写一个高效的算法来搜索 `m x n` 矩阵 `matrix` 中的一个目标值 `target` 。该矩阵具有以下特性：

- 每行的元素从左到右升序排列。
- 每列的元素从上到下升序排列。

思路：选一个**起点**：**右上角**！因为右上角是**特殊位置**，它是这一行最大的数，它是这一列最小的数，**比 target 大 → 向左走（变小）**，**比 target 小 → 向下走（变大）**

```python
class Solution:
    def searchMatrix(self, matrix: List[List[int]], target: int) -> bool:
        m, n = len(matrix), len(matrix[0])
        left, right = 0, n - 1
        while left < m and right >= 0:
            if matrix[left][right] == target:
                return True
            if matrix[left][right] > target:
                right -= 1
            else:
                left += 1
        return False
```

时间复杂度：`O (m + n)`

#### 206.反转链表

题目：给你单链表的头节点 `head` ，请你反转链表，并返回反转后的链表。

```python
        head.next = None
        return new_head
```

```python
            slow = slow.next
            fast = fast.next.next
        return True
```

如果链表中有某个节点，可以通过连续跟踪 `next` 指针再次到达，则链表中存在环。 为了表示给定链表中的环，评测系统内部使用整数 `pos` 来表示链表尾连接到链表中的位置（**索引从 0 开始**）。如果 `pos` 是 `-1`，则在该链表中没有环。**注意：`pos` 不作为参数进行传递**，仅仅是为了标识链表的实际情况。

**不允许修改** 链表。

思路：**快慢指针先相遇**（证明有环），把慢指针放回链表头，两个指针现在都一次走 1 步，再次相遇的点，就是入环的第一个节点，头节点走到入环点的距离 = 相遇点走到入环点的距离

```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None
# 哈希表
class Solution:
    def detectCycle(self, head: Optional[ListNode]) -> Optional[ListNode]:
        vertex = set()
        while head:
            if head in vertex:
                return head
            vertex.add(head)
            head = head.next
        return None
# 时间复杂度：O(n)
# 空间复杂度：O(n)

class Solution:

        fast = head
        # 第一步：快慢指针，找到相遇点

            slow = slow.next
            fast = fast.next.next
            # 相遇了！说明有环
            if slow == fast:
                # 第二步：把一个指针放回起点，同步走，再次相遇就是入环点

                while slow != fast:
                    slow = slow.next
                    fast = fast.next
                return slow  # 入环第一个节点
        # 无环
        return None

# 空间复杂度：O(1)
```

#### 2.两数相加

题目：给你两个 **非空** 的链表，表示两个非负的整数。它们每位数字都是按照 **逆序** 的方式存储的，并且每个节点只能存储 **一位** 数字。

请你将两个数相加，并以相同形式返回一个表示和的链表。

你可以假设除了数字 0 之外，这两个数都不会以 0 开头。

思路：把两个**逆序存储**的链表，像**列竖式加法**一样，**从左到右逐位相加 → 记录进位 → 生成新链表**

```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def addTwoNumbers(self, l1: Optional[ListNode], l2: Optional[ListNode]) -> Optional[ListNode]:
        # 创建一个虚拟头节点，方便操作
        dummy = ListNode(0)
        current = dummy  # current 指针用来构建新链表
        carry = 0        # 进位初始化为 0
        # 只要 l1 或 l2 还有节点，或者还有进位没处理完，就继续循环
        while l1 or l2 or carry:
            # 如果节点为空，则该位的值默认为 0
            val1 = l1.val if l1 else 0
            val2 = l2.val if l2 else 0
            # 计算当前位的和 + 之前的进位
            total_sum = val1 + val2 + carry
            # 更新进位 (整除 10)
            carry = total_sum // 10
            # 当前位实际存储的数字 (对 10 取余)
            new_val = total_sum % 10
            # 创建新节点并连接
            current.next = ListNode(new_val)
            current = current.next
            # 移动指针到下一位 (如果存在)
            if l1: l1 = l1.next
            if l2: l2 = l2.next
        return dummy.next
# 时间复杂度：O(max (n,m))
# 空间复杂度：O(max (n,m))
```

#### 19.删除链表的倒数第N个结点

```python
            first = first.next
            second = second.next
```

148.排序链表

```python
    def sortList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        def sortFunc(head: ListNode, tail: ListNode) -> ListNode:
            if not head:
                return head
            if head.next == tail:
                head.next = None
                return head
            slow = fast = head
            while fast != tail:
                slow = slow.next
                fast = fast.next
                if fast != tail:
                    fast = fast.next
            mid = slow
            return merge(sortFunc(head, mid), sortFunc(mid, tail))
        def merge(head1: ListNode, head2: ListNode) -> ListNode:
            dummyHead = ListNode(0)
            temp, temp1, temp2 = dummyHead, head1, head2
            while temp1 and temp2:
                if temp1.val <= temp2.val:
                    temp.next = temp1
                    temp1 = temp1.next
                else:
                    temp.next = temp2
                    temp2 = temp2.next
                temp = temp.next
            if temp1:
                temp.next = temp1
            elif temp2:
                temp.next = temp2
            return dummyHead.next
        return sortFunc(head, None)
```

#### 146.LRU缓存

题目：请你设计并实现一个满足 LRU (最近最少使用) 缓存 约束的数据结构。

实现 `LRUCache` 类：

- `LRUCache(int capacity)` 以 **正整数** 作为容量 `capacity` 初始化 LRU 缓存
- `int get(int key)` 如果关键字 `key` 存在于缓存中，则返回关键字的值，否则返回 `-1` 。
- `void put(int key, int value)` 如果关键字 `key` 已经存在，则变更其数据值 `value` ；如果不存在，则向缓存中插入该组 `key-value` 。如果插入操作导致关键字数量超过 `capacity` ，则应该 **逐出** 最久未使用的关键字。

函数 `get` 和 `put` 必须以 `O(1)` 的平均时间复杂度运行。

思路：Python自带有序字典的方法**`move_to_end(key)`**把某个 key 移动到**字典最后面（最新端）**，**`popitem(last=False)`**删除 **最前面（最久没使用）** 的元素，不加 self → 临时变量，用完就没加 self → 属于对象，一直存在

```python
class LRUCache(OrderedDict):

    def __init__(self, capacity: int):
        self.capacity = capacity

    def get(self, key: int) -> int:
        if key not in self:
            return -1
        self.move_to_end(key)
        return self[key]

    def put(self, key: int, value: int) -> None:
        if key in self:
            self.move_to_end(key)
        self[key] = value
        if len(self) > self.capacity:
            self.popitem(last = False)
# 时间复杂度：O(1)
```

#### 94.二叉树的中序遍历

题目：给定一个二叉树的根节点 `root` ，返回 *它的 **中序** 遍历* 。

思路：递归 + 中序规则（左→根→右），分治拆解树结构，终止于空节点

```python
class Solution:
    def inorderTraversal(self, root: Optional[TreeNode]) -> List[int]:
        res = []
        def traversal(node):
            if not node:
                return
            # 第一步：递归遍历左子树
            traversal(node.left)
            # 第二步：访问当前根节点，把值加入结果
            res.append(node.val)
            traversal(node.right)
        traversal(root)
        return res
```

空间复杂度：**O(n)**

#### 104.二叉树的最大深度

题目：给定一个二叉树 `root` ，返回其最大深度。二叉树的 **最大深度** 是指从根节点到最远叶子节点的最长路径上的节点数。

思路：层序遍历（BFS），二叉树的最大深度 = 树的总层数，BFS 的核心逻辑：**一层一层往下遍历**，每遍历完一层，深度 +1，直到没有下一层节点为止

```python
# 定义二叉树节点的结构
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def maxDepth(self, root: Optional[TreeNode]) -> int:
        if not root:
            return  0
        level_nodes = [root]
        # 定义递归函数实现BFS
        def bfs(nodes):
            if not nodes: # 递归终止条件：当前层没有节点了
                return 0
            # 收集下一层的所有节点
            next_level = []
            for node in nodes:
                # 如果这个节点有左孩子，就加入下一层
                if node.left:
                    next_level.append(node.left)
                # 如果这个节点有右孩子，就加入下一层
                if node.right:
                    next_level.append(node.right)
            # 递归处理下一层 + 当前层的深度1
            return 1 + bfs(next_level)
        return bfs(level_nodes)
```

时间复杂度：**O(n)**

空间复杂度：**O(n)**

#### 226.翻转二叉树

题目：给你一棵二叉树的根节点 `root` ，翻转这棵二叉树，并返回其根节点。

思路：**后序遍历**逻辑，**左 → 右 → 根**（先处理完左右孩子，最后处理自己），递归执行顺序：**从下往上**

1. **一路到底**：代码先不交换，一直递归走到**最底层叶子节点**；
2. **从底开始翻转**：先翻转**最后一层**节点的左右（空节点交换还是空）；
3. **逐层向上**：处理完底层，返回上一层交换，**一层层往上翻**；
4. **最后翻根节点**：直到回到根节点，交换它的左右，完成整棵树翻转。

```python
class Solution:
    def invertTree(self, root: Optional[TreeNode]) -> Optional[TreeNode]:
        if not root:
            return root
        # 把左子树完全翻转好，存到 left 变量里
        left = self.invertTree(root.left)
        # 把右子树完全翻转好，存到 right 变量里
        right =  self.invertTree(root.right)
        # 第三步：交换 当前节点的 左孩子 和 右孩子
        # 左边 = 翻转好的右子树，右边 = 翻转好的左子树
        root.left, root.right = right, left
        return root
```

时间复杂度：**O(n)**

空间复杂度：**O(n)**

#### 101.对称二叉树

题目：给你一个二叉树的根节点 `root` ， 检查它是否轴对称。

思路：一棵二叉树**对称** = **根的左子树 和 右子树 互为镜像**

互为镜像的规则：

1. **两个节点的值必须一样**
2. **左节点的左孩子 ↔ 右节点的右孩子**
3. **左节点的右孩子 ↔ 右节点的左孩子**

就是**里外翻转对应相等**。

```python
class Solution:
    def isSymmetric(self, root: Optional[TreeNode]) -> bool:
        if root is None:
            return False
        # 定义递归函数：判断两个节点 p 和 q 是否镜像对称
        def istrue(p:Optional[TreeNode],q:Optional[TreeNode])->bool:
            # 情况1：其中一个是空，另一个不是 → 不对称
            # 情况2：两个都空 → 对称
            # 这句代码直接同时判断了这两种情况
            if q is None or p is None:
                return p is q
            # 要满足对称，必须同时满足 3 个条件：
            # 1. 当前两个节点值相等
            # 2. p 的左孩子 和 q 的右孩子 对称
            # 3. p 的右孩子 和 q 的左孩子 对称
            return p.val==q.val and istrue(p.left,q.right) and istrue(p.right,q.left)
        return istrue(root.left,root.right)
```

时间复杂度：**O(n)**

空间复杂度：**O(n)**

#### 543.二叉树的直径

题目：给你一棵二叉树的根节点，返回该树的 **直径** 。

二叉树的 **直径** 是指树中任意两个节点之间最长路径的 **长度** 。这条路径可能经过也可能不经过根节点 `root` 。

两节点之间路径的 **长度** 由它们之间边数表示。

思路：边的条数 = 节点总数 - 1，递归算深度 → 顺便算每个节点的左右深度和 → 取最大值就是直径

```python
class Solution:
    def diameterOfBinaryTree(self, root: Optional[TreeNode]) -> int:
        self.ans = 0
        # 定义递归函数：求以 node 为根的子树【最大深度】
        def depth(node):
            if not node:
                return 0
            # 递归计算：左子树的最大深度 L
            L = depth(node.left)
            # 递归计算：右子树的最大深度 R
            R = depth(node.right)
            # 【核心！】经过当前节点的直径 = 左深度 + 右深度 + 1（当前节点）
            # 更新全局最大直径（存的是节点个数）
            self.ans = max(self.ans, L + R + 1)
            # 返回：当前节点的最大深度 = 左右最大深度 + 1
            return max(L, R) + 1
        depth(root)
        return self.ans - 1
```

时间复杂度：**O(n)**

空间复杂度：**O(n)**

#### 102.二叉树的层序遍历

题目：给你二叉树的根节点 `root` ，返回其节点值的 **层序遍历** 。 （即逐层地，从左到右访问所有节点）。

思路：**标准层序遍历（BFS）**，**当前层 → 收集值 → 找下一层 → 循环直到结束**

```python
class Solution:
    def levelOrder(self, root: Optional[TreeNode]) -> List[List[int]]:
        if not root:
            return []
        result = []
        # 当前正在遍历的这一层的所有节点（一开始只有根节点）
        current_nodes = [root]
        while current_nodes:
            next_nodes = []
            current_nodes_vals = []
            for node in current_nodes:
                # 把当前节点的值加入当前层的值列表
                current_nodes_vals.append(node.val)
                if node.left:
                    next_nodes.append(node.left)
                if node.right:
                    next_nodes.append(node.right)
            # 当前层遍历完了，把这一层的值放进结果里
            result.append(current_nodes_vals)
            current_nodes = next_nodes
        return result
```

时间复杂度：**O(n)**

空间复杂度：**O(n)**

#### 108.将有序数组转换为二叉搜索树

题目：给你一个整数数组 `nums` ，其中元素已经按 **升序** 排列，请你将其转换为一棵 平衡 二叉搜索树。

**平衡二叉树** 是指该树所有节点的左右子树的高度相差不超过 1。

思路：有序数组要变成**平衡 BST**，必须让**左右子树节点数量差不多**，所以每次都拿**中间元素当根节点**，取中间当根 → 左边建左子树 → 右边建右子树 → 递归完成平衡 BST

```python
class Solution:
    def sortedArrayToBST(self, nums: List[int]) -> Optional[TreeNode]:
        # 定义递归辅助函数：
        # left, right 表示当前正在处理的数组区间 [left, right]
        def helper(left, right):
            # 递归终止条件：区间不合法，左边 > 右边 → 没有节点可以创建，返回空
            if left > right:
                return None
            # 1. 取当前区间的【中间位置】作为根节点
            # 这样才能保证树是【高度平衡】的（左右子树高度差不超过1）
            mid = (left + right) // 2
            # 2. 创建根节点，值 = 数组中间的数字
            root = TreeNode(nums[mid])
            # 3. 递归构建【左子树】：数组区间 [left, mid-1]
            root.left = helper(left, mid - 1)
            # 4. 递归构建【右子树】：数组区间 [mid+1, right]
            root.right = helper(mid + 1 ,right)
            return root
        return helper(0, len(nums) - 1)
```

**时间复杂度：O (n)**，每个元素创建一个节点，只访问一次

**空间复杂度：O (log n)**，递归深度是树高，平衡树高度 = log n

#### 98.验证二叉树

题目：给你一个二叉树的根节点 `root` ，判断其是否是一个有效的二叉搜索树。

**有效** 二叉搜索树定义如下：

- 节点的左子树只包含 **严格小于** 当前节点的数。
- 节点的右子树只包含 **严格大于** 当前节点的数。
- 所有左子树和右子树自身必须也是二叉搜索树。

思路：定义 `helper(node, lower, upper)` 递归函数，校验当前节点 `node` 是否在 `(lower, upper)` 的取值范围内**右子树**：右子树所有节点必须 **> 当前节点值**，所以更新下界为 `val`，上界保持 `upper`，**左子树**：左子树所有节点必须 **< 当前节点值**，所以更新上界为 `val`，下界保持 `lower`

```python
class Solution:
    def isValidBST(self, root: Optional[TreeNode]) -> bool:
        def helper(node, lower = float('-inf'), upper = float('inf')) -> bool:
            if not node:
                return True
            val = node.val
            if val <= lower or val >= upper:
                return False
            if not helper(node.right, val, upper):
                return False
            if not helper(node.left, lower, val):
                return False
            return True
        return helper(root)
# 时间复杂度：O(n)
# 空间复杂度：O(n)
```

```python
class Solution:
    def kthSmallest(self, root: Optional[TreeNode], k: int) -> int:
        stack = []
        while root or stack:
            while root:
                stack.append(root)
                root = root.left
            root = stack.pop()
            k -= 1
            if k == 0:
                return root.val
            root = root.right
# 时间复杂度：O(H+k)，其中 H 是树的高度。在开始遍历之前，我们需要 O(H) 到达叶结点。当树是平衡树时，时间复杂度取得最小值 O(logN+k)；当树是线性树（树中每个结点都只有一个子结点或没有子结点）时，时间复杂度取得最大值 O(N+k)
# 空间复杂度：O(H)，栈中最多需要存储 H 个元素。当树是平衡树时，空间复杂度取得最小值 O(logN)；当树是线性树时，空间复杂度取得最大值 O(N)
```

```python
class Solution:

            return []

            for i in range(nums):
                node = queue.pop(0)
                if i == nums - 1:
                    result.append(node.val)
                if node.left:

                if node.right:
                    queue.append(node.right)
```

- 展开后的单链表应该与二叉树 先序遍历 顺序相同。

```python
class Solution:
    def flatten(self, root: Optional[TreeNode]) -> None:
        """
        Do not return anything, modify root in-place instead.
        """
        curr = root
        while curr:
            if curr.left:
                predecessor = nxt = curr.left
                while predecessor.right:
                    predecessor = predecessor.right
                predecessor.right = curr.right
                curr.left = None
                curr.right = nxt
            curr = curr.right
# 空间复杂度：O(1)（最优）
```


#### 437.路径总和II

题目：给定一个二叉树的根节点 `root` ，和一个整数 `targetSum` ，求该二叉树里节点值之和等于 `targetSum` 的 **路径** 的数目。

```python
class Solution:
    def pathSum(self, root: Optional[TreeNode], targetSum: int) -> int:
        prefix = collections.defaultdict(int)
        prefix[0] = 1
        def dfs(root, curr):
            if not root:
                return 0
            ret = 0
            curr += root.val
            ret += prefix[curr - targetSum]
            prefix[curr] += 1
            ret += dfs(root.left, curr)
            ret += dfs(root.right, curr)
            prefix[curr] -= 1
            return ret
        return dfs(root, 0)
```



#### 236.二叉树的最近公共祖先

```python
class Solution:

            return root
        left = self.lowestCommonAncestor(root.left, p, q)

            return right
        return None
```

题目：给你一个由 `'1'`（陆地）和 `'0'`（水）组成的的二维网格，请你计算网格中岛屿的数量。岛屿总是被水包围，并且每座岛屿只能由水平方向和/或竖直方向上相邻的陆地连接形成。此外，你可以假设该网格的四条边均被水包围。

思路：采用深度优先搜索（DFS），遍历二维网格，遇到未访问的陆地`1`时，岛屿数量加 1，并通过 DFS 将其上下左右所有相连的陆地全部标记为水域`0`（避免重复统计），最终统计出总岛屿数。

```python
class Solution:
    def dfs(self, grid, r, c):
        grid[r][c] = 0
        nr = len(grid)
        nc = len(grid[0])
        for x, y in [(r - 1, c), (r + 1, c), (r, c - 1), (r, c + 1)]:
            if 0 <= x < nr and 0 <= y < nc and grid[x][y] == "1":
                self.dfs(grid, x, y)
    def numIslands(self, grid: List[List[str]]) -> int:
        nr = len(grid)
        if nr == 0:
            return 0
        nc = len(grid[0])
        num_islands = 0
        for r in range(nr):
            for c in range(nc):
                if grid[r][c] == "1":
                    num_islands += 1
                    self.dfs(grid, r, c)
        return nmu_islands
```

时间复杂度：**O(m×n)**（m 为行数，n 为列数，每个格子仅访问一次）

- 值 `0` 代表空单元格；
- 值 `1` 代表新鲜橘子；
- 值 `2` 代表腐烂的橘子。

每分钟，腐烂的橘子 **周围 4 个方向上相邻** 的新鲜橘子都会腐烂。

返回 直到单元格中没有新鲜橘子为止所必须经过的最小分钟数。如果不可能，返回 `-1` 。

思路：采用**BFS 广度优先搜索**（多源队列），先收集所有初始腐烂橘子入队，统计新鲜橘子数量；每一轮遍历队列中所有当前腐烂橘子，向四向扩散感染新鲜橘子，将其变为腐烂并加入队列，每轮记一分钟；直到无新鲜橘子或无腐烂橘子可扩散，最终新鲜橘子全烂则返回时间，否则返回 - 1

```python
class Solution:
    def orangesRotting(self, grid: List[List[int]]) -> int:
        rows, cols = len(grid), len(grid[0])
        # 初始化队列，用于BFS广度优先搜索（存储腐烂橘子的坐标）
        queue = collections.deque()
        fresh_count = 0
        for r in range(rows):
            for c in range(cols):
                if grid[r][c] == 2:
                    queue.append((r, c))
                elif grid[r][c] == 1:
                    fresh_count += 1
        if fresh_count == 0:
            return 0
        minutes = 0
        # BFS循环：队列不为空 且 还有新鲜橘子
        while queue and fresh_count > 0:
            minutes += 1
            # 一次性处理当前队列里所有腐烂橘子（一层）
            for _ in range(len(queue)):
                r, c = queue.popleft()
                for dr, dc in [(1,0), (-1,0), (0,1), (0,-1)]:
                    nr, nc = r + dr, c + dc
                    if 0 <= nr < rows and 0 <= nc < cols and grid[nr][nc] == 1:
                        grid[nr][nc] = 2
                        fresh_count -= 1
                        queue.append((nr, nc))
        return minutes if fresh_count == 0 else -1
```

时间复杂度：O(nm)。即进行一次广度优先搜索的时间，其中 n,m 分别为 grid 的行数与列数。

空间复杂度：O(nm)。需要额外的 dis 数组记录每个新鲜橘子被腐烂的最短时间，大小为 O(nm)，且广度优先搜索中队列里存放的状态最多不会超过 nm 个，最多需要 O(nm) 的空间，所以最后的空间复杂度为 O(nm)。

#### 207.课程表

题目：你这个学期必须选修 `numCourses` 门课程，记为 `0` 到 `numCourses - 1` 。

在选修某些课程之前需要一些先修课程。 先修课程按数组 `prerequisites` 给出，其中 `prerequisites[i] = [ai, bi]` ，表示如果要学习课程 `ai` 则 **必须** 先学习课程 `bi` 。

- 例如，先修课程对 `[0, 1]` 表示：想要学习课程 `0` ，你需要先完成课程 `1` 。

请你判断是否可能完成所有课程的学习？如果可以，返回 `true` ；否则，返回 `false` 。

思路：这是**拓扑排序（BFS 版）**，用于判断课程依赖图是否**有环**（无环就能上完所有课）。

1. 先构建课程有向图，并统计每门课的**入度（前置课程数）**。
2. 把**入度为 0**（无前置课）的课程加入队列，开始学习。
3. 每学完一门课，就减少它后续课程的入度；当某门课入度变为 0，就加入队列学习。
4. 最终**能学完的课程数 == 总课程数**，说明无环，返回 True；否则有环，返回 False。

```python
class Solution:
    def canFinish(self, numCourses: int, prerequisites: List[List[int]]) -> bool:
        # 邻接表：存储课程依赖图
        graph = defaultdict(list)
        # 入度数组：记录每门课需要的前置课程数量
        indegree = [0] * numCourses
        # 构建图 + 统计入度
        # dest: 目标课, src: 前置课 (先学src才能学dest)
        for dest, src in prerequisites:
            graph[src].append(dest)
            indegree[dest] += 1
        # 初始化队列：所有入度为0（无前置课）的课程
        queue = deque([i for i in range(numCourses) if indegree[i] == 0])
        learned_course = 0
        # BFS拓扑排序
        while queue:
            course = queue.popleft()
            learned_course += 1
            for next_course in graph[course]:
                indegree[next_course] -= 1
                if indegree[next_course] == 0:
                    queue.append(next_course)
        # 能学完所有课 = 无环 = 返回Tru
        return learned_course == numCourses
```

时间复杂度: *O*(*n*+*m*)，其中 *n* 为课程数，*m* 为先修课程的要求数。这其实就是对图进行深度优先搜索的时间复杂度。

空间复杂度: *O*(*n*+*m*)

#### 208.实现Trie（前缀树）

题目：**[Trie](https://baike.baidu.com/item/字典树/9825209?fr=aladdin)**（发音类似 "try"）或者说 **前缀树** 是一种树形数据结构，用于高效地存储和检索字符串数据集中的键。这一数据结构有相当多的应用情景，例如自动补全和拼写检查。

请你实现 Trie 类：

- `Trie()` 初始化前缀树对象。
- `void insert(String word)` 向前缀树中插入字符串 `word` 。
- `boolean search(String word)` 如果字符串 `word` 在前缀树中，返回 `true`（即，在检索之前已经插入）；否则，返回 `false` 。
- `boolean startsWith(String prefix)` 如果之前已经插入的字符串 `word` 的前缀之一为 `prefix` ，返回 `true` ；否则，返回 `false` 。

思路：这是字典树（Trie）的标准实现，专门高效存储和查询字符串：

1. 每个节点存 26 个字母子节点 + 标记是否为单词结尾。
2. **insert**：逐字符遍历，不存在则新建节点，走到最后标记单词结束。
3. **search**：按字符走到末尾，若节点存在且标记结束则返回 True。
4. **startsWith**：只需按前缀能走到末尾就返回 True，不要求是完整单词。

```python
class Trie:

    def __init__(self):
        self.children = [None] * 26
        self.isEnd = False

    def searchPrefix(self, prefix: str) -> "Trie":
        node = self
        for ch in prefix:
            ch = ord(ch) - ord("a")
            if not node.children[ch]:
                return None
            node = node.children[ch]
        return node

    def insert(self, word: str) -> None:
        node = self
        for ch in word:
            ch =ord(ch) - ord("a")
            if not node.children[ch]:
                node.children[ch] = Trie()
            node = node.children[ch]
        node.isEnd = True

    def search(self, word: str) -> bool:
        node = self.searchPrefix(word)
        return node is not None and node.isEnd

    def startsWith(self, prefix: str) -> bool:
        return self.searchPrefix(prefix) is not None
```

时间复杂度：插入 / 搜索 / 前缀查询均为 **O(L)**，L 是字符串长度。

空间复杂度：**O(26×L)**，L 为所有存入字符串总字符数。

#### 46.全排列

题目：给定一个不含重复数字的数组 `nums` ，返回其 *所有可能的全排列* 。你可以 **按任意顺序** 返回答案。

思路：逐个选择未使用的数字构建路径，构建完成就保存，然后回退一步尝试其他选择，直到遍历完所有可能

```python
class Solution:
    def permute(self, nums: List[int]) -> List[List[int]]:
        result = []
        # 记录当前正在拼接的一条排列路径
        path = []
        # 标记数组，记录哪些数字已经被使用（避免重复选）
        used = [False] * len(nums)
        def backtrack():
            if len(path) == len(nums):
                result.append(path.copy())
                return
            for i in range(len(nums)):
                if used[i]:
                    continue
                # 把数字加入路径，标记为已使用
                path.append(nums[i])
                used[i] = True
                backtrack()
                path.pop()
                used[i] = False
        backtrack()
        return result
```

时间复杂度：`O (n × n!)`，n是数组长度，**生成所有排列需要 n! 次递归**，第一层 n 种选择，第二层 n-1 种…… 最后 1 种 → 总路径数 = n!，**每次找到完整排列需要 O (n) 时间**，要把长度为 n 的 path 复制到结果里，复制操作是 O (n)

空间复杂度：`O (n)`

#### 78.子集

题目：给你一个整数数组 `nums` ，数组中的元素 **互不相同** 。返回该数组所有可能的子集（幂集）。解集 **不能** 包含重复的子集。你可以按 **任意顺序** 返回解集。

思路：和全排列不同，**子集不关心顺序，只关心选不选**，用 **start 索引** 控制不回头选、不重复，从前往后选元素，每一步都记录当前子集，不回头选，生成所有可能组合

```python
class Solution:
    def subsets(self, nums: List[int]) -> List[List[int]]:
        result = []
        path = []
        def backtrack(start):
            result.append(path.copy())
            for i in range(start, len(nums)):
                path.append(nums[i])
                backtrack(i + 1)
                path.pop()
        backtrack(0)
        return result
```

时间复杂度：`O (n × 2ⁿ)`

空间复杂度：`O(n)`

#### 39.组合总和

题目：给你一个 **无重复元素** 的整数数组 `candidates` 和一个目标整数 `target` ，找出 `candidates` 中可以使数字和为目标数 `target` 的 所有 **不同组合** ，并以列表形式返回。你可以按 **任意顺序** 返回这些组合。

`candidates` 中的 **同一个** 数字可以 **无限制重复被选取** 。如果至少一个数字的被选数量不同，则两种组合是不同的。 

对于给定的输入，保证和为 `target` 的不同组合数少于 `150` 个。

```python
class Solution:
    def combinationSum(self, candidates: List[int], target: int) -> List[List[int]]:
        result = []
        candidates.sort()
        def backtrack(remain, path, start):
            # remain 记录剩余需要匹配的数值, path 记录当前已匹配结果, start 记录起始索引避免往回找
            if remain == 0 :
                result.append(path.copy())
                return
            for i in range(start, len(candidates)):
                num = candidates[i]
                if num > remain:
                    break
                path.append(num)
                backtrack(remain - num, path, i)
                path.pop()
        backtrack(target, [], 0)
        return result
```

#### 22.括号生成

题目：数字 `n` 代表生成括号的对数，请你设计一个函数，用于能够生成所有可能的并且 **有效的** 括号组合。

思路：左括号 `(` 数量 **不能超过 n**们，右括号 `)` 数量 **不能超过左括号**，最终长度必须是 **2n**，只在合法位置加括号，左括号优先，右括号不超过左括号，生成所有有效组合。

```python
class Solution:
    def generateParenthesis(self, n: int) -> List[str]:
        result = []
        def backtrack(path, open_cnt, close_cnt):
            if len(path) == 2 * n:
                result.append(''.join(path))
                return
            if open_cnt < n:
                path.append('(')
                backtrack(path, open_cnt + 1, close_cnt)
                path.pop()
            if close_cnt < open_cnt:
                path.append(')')
                backtrack(path, open_cnt, close_cnt + 1)
                path.pop()
        backtrack([], 0, 0)
        return result
```

时间复杂度：O(4ⁿ / √n)

空间复杂度：`O(N)`

#### 79.单词搜索

题目：给定一个 `m x n` 二维字符网格 `board` 和一个字符串单词 `word` 。如果 `word` 存在于网格中，返回 `true` ；否则，返回 `false` 。

单词必须按照字母顺序，通过相邻的单元格内的字母构成，其中“相邻”单元格是那些水平相邻或垂直相邻的单元格。同一个单元格内的字母不允许被重复使用。

```python
class Solution:
    def exist(self, board: List[List[str]], word: str) -> bool:
        rows = len(board)
        cols = len(board[0])
        directions = [(-1, 0), (1, 0), (0, 1), (0, -1)]
        visited = [[False] * cols for _ in range(rows)] 
        def backtrack(i, j, path):
            if path == len(word):
                return True
            for dx, dy in directions:
                x = i + dx
                y = j + dy
                if 0 <= x < rows and 0 <= y < cols and not visited[x][y] and board[x][y] == word[path]:
                    visited[x][y] = True
                    if backtrack(x, y, path + 1):
                        return True
                    visited[x][y] = False
        for i in range(rows):
            for j in range(cols):
                if board[i][j] == word[0]:
                    visited[i][j] = True
                    if backtrack(i, j, 1):
                        return True
                    visited[i][j] = False
        return False
```

#### 131.分割回文串

题目：给你一个字符串 `s`，请你将 `s` 分割成一些 子串，使每个子串都是 **回文串** 。返回 `s` 所有可能的分割方案。

思路：**回溯分割 + 动态规划预处理回文**，提前用 DP 存所有回文子串，再用回溯从左到右分割，只选合法回文段

```python
class Solution:
    def partition(self, s: str) -> List[List[str]]:
        result = []
        n = len(s)
        # DP 预处理（提前算好所有回文子串），建二维数组 dp[i][j]：表示 s[i...j] 是否是回文
        dp = [[True] * n for _ in range(n)]
        for i in range(n-1, -1, -1):
            for j in range(i+1, n):
                dp[i][j] = (s[i] == s[j]) and dp[i+1][j-1]
        def backtrack(path, start):
            if start == n:
                result.append(path.copy())
                return
            for end in range(start, n):
                if dp[start][end]:
                    path.append(s[start:end+1])
                    backtrack(path, end+1)
                    path.pop()
        backtrack([], 0)
        return result
```

时间复杂度：`O(2ⁿ)`

空间复杂度：`O(n²)`



回溯的本质是：**做选择 → 递归探索 → 撤销选择**，核心是通过「试错」的方式遍历所有可能的解空间。

我先给你一个通用的伪代码模板，再拆解每个部分的作用：

```python
def backtrack(参数):
    # 1. 终止条件：找到一个合法解，记录结果
    if 满足终止条件:
        把当前路径加入结果集
        return
    
    # 2. 遍历所有可选的「选择」
    for 选择 in 选择列表:
        # 剪枝：排除不合法/无效的选择（可选，但能大幅提升效率）
        if 选择不合法:
            continue/break
        
        # 3. 做出选择：把当前选择加入路径
        路径.append(选择)
        
        # 4. 递归探索：进入下一层决策树
        backtrack(更新后的参数)
        
        # 5. 撤销选择：回溯，回到上一步（关键！）
        路径.pop()

# 主函数：初始化结果集和路径，调用回溯函数
def main(问题参数):
    结果集 = []
    路径 = []
    backtrack(初始参数)
    return 结果集
```

### 模板核心拆解（5 个关键部分）

我用「人话 + 对应组合总和案例」帮你理解每个部分的作用：

| 模板部分 |              核心作用              |             组合总和中的对应实现             |
| :------: | :--------------------------------: | :------------------------------------------: |
| 终止条件 |  划定递归的边界，找到有效解时记录  | `if remain == 0: result.append(path.copy())` |
| 遍历选择 |      遍历当前层所有可做的选择      |   `for i in range(start, len(candidates))`   |
|   剪枝   |    提前排除无效选择，避免走弯路    |    `if num > remain: break`（排序后剪枝）    |
|  做选择  | 将当前选择加入「路径」（记录已选） |              `path.append(num)`              |
| 递归探索 |       带着新的状态进入下一层       |      `backtrack(remain - num, path, i)`      |
| 撤销选择 |      回到上一层，尝试其他选择      |                 `path.pop()`                 |

### 模板的灵活适配（不同题型的调整点）

你只需要根据题目的**核心约束**，调整模板中的「选择列表」和「参数」即可，我用常见题型举例：

#### 1. 组合类问题（如组合总和、组合总和 II、组合）

- **核心约束**：元素可重复选 / 不可重复选、组合不重复（[2,3] 和 [3,2] 算同一个）

- **调整点**：

  - 用 `start` 参数控制「选择列表的起始位置」（避免重复组合）；
  - 可重复选：`backtrack(..., i)`（下一层仍从当前索引开始）；
  - 不可重复选：`backtrack(..., i+1)`（下一层从下一个索引开始）。


#### 2. 排列类问题（如全排列、全排列 II）

- **核心约束**：元素可重复 / 不可重复、排列要去重

- **调整点**：

  - 不用 `start` 参数，而是用「used 数组」标记已选元素；
  - 去重：排序后跳过重复元素 + 检查 `used` 数组。


#### 3. 子集类问题（如子集、子集 II）

- **核心约束**：收集所有子集、子集不重复

- **调整点**：

  - 终止条件不是「凑够目标」，而是「遍历完所有元素」（或每一步都把路径加入结果）；
  - 用 `start` 参数控制起始位置，避免重复子集。


#### 4. 棋盘类问题（如 N 皇后、解数独）

- **核心约束**：满足棋盘规则（如皇后不冲突）
- **调整点**：
  - 「选择列表」是当前行的所有列；
  - 「剪枝」是检查当前位置是否符合规则（如皇后是否冲突）；
  - 终止条件是「遍历完所有行」。

#### 35.搜索插入位置

题目：给定一个排序数组和一个目标值，在数组中找到目标值，并返回其索引。如果目标值不存在于数组中，返回它将会被按顺序插入的位置。请必须使用时间复杂度为 `O(log n)` 的算法。

思路：对有序数组使用**二分查找**，找到目标值则返回索引，未找到则循环结束时`left`即为插入位置

```python
class Solution:
    def searchInsert(self, nums: List[int], target: int) -> int:
        left = 0
        right = len(nums) - 1
        while left <= right:
            middle = left + (right - left) // 2
            if nums[middle] == target:
                return middle
            elif nums[middle] < target:
                left = middle + 1
            else:
                right = middle - 1
        return left
```

时间复杂度： O (log n)

空间复杂度： O (1)

```python
class Solution:
    def searchMatrix(self, matrix: List[List[int]], target: int) -> bool:
        if not matrix or not matrix[0]:
            return False
        m = len(matrix)
        n = len(matrix[0])
        left = 0
        right = m * n - 1
        while left <= right:
            mid = left + (right - left) // 2
            row = mid // n
            col = mid % n
            current = matrix[row][col]
            if current == target:
                return True
            elif current < target:
                left = mid + 1
            else:
                right = mid - 1
        return False
```

时间复杂度：**O(log(mn))**

空间复杂度：**O(1)**

#### 34.在排序数组中查找元素的第一个和最后一个位置

题目：给你一个按照非递减顺序排列的整数数组 `nums`，和一个目标值 `target`。请你找出给定目标值在数组中的开始位置和结束位置。

如果数组中不存在目标值 `target`，返回 `[-1, -1]`。

你必须设计并实现时间复杂度为 `O(log n)` 的算法解决此问题。

思路：用**两次二分查找**：第一次找**目标值左边界**，第二次找**目标值右边界**，最后校验边界有效性得到结果

```python
class Solution:
    def searchRange(self, nums: List[int], target: int) -> List[int]:
        def find_left():
            left, right = 0, len(nums) - 1
            res = len(nums)
            while left <= right:
                mid = (left + right) // 2
                if nums[mid] >= target:
                    res = mid
                    right = mid - 1
                else:
                    left = mid + 1
            return res
        def find_right():
            left, right = 0, len(nums) - 1
            res = -1
            while left <= right:
                mid = (left + right) // 2
                if nums[mid] <= target:
                    res = mid
                    left = mid + 1
                else:
                    right = mid - 1
            return res
        left = find_left()
        right = find_right()
        if left > right or left >= len(nums) or nums[left] != target:
            return [-1, -1]
        return [left, right]
```

时间复杂度：**O(log n)**

空间复杂度：**O(1)**

给你 **旋转后** 的数组 `nums` 和一个整数 `target` ，如果 `nums` 中存在这个目标值 `target` ，则返回它的下标，否则返回 `-1` 。

你必须设计一个时间复杂度为 `O(log n)` 的算法解决此问题。

 思路：对**旋转有序数组**做二分查找，每次判断**左 / 右哪一段是有序的**，再看目标值是否在有序区间内，以此缩小区间查找

```python
class Solution:
    def search(self, nums: List[int], target: int) -> int:
        left, right = 0, len(nums) - 1
        while left <= right:
            mid = (left + right) // 2
            if nums[mid] == target:
                return mid
            if nums[left] <= nums[mid]:
                if nums[left] <= target < nums[mid]:
                    right = mid - 1
                else:
                    left = mid + 1
            else:
                if nums[mid] < target <= nums[right]:
                    left = mid + 1
                else:
                    right = mid - 1
        return -1
```

时间复杂度：**O(log n)**

空间复杂度：**O(1)**

```python
class Solution:

            else:
                left = mid + 1
```

#### 155.最小栈

题目：设计一个支持 `push` ，`pop` ，`top` 操作，并能在常数时间内检索到最小元素的栈。

实现 `MinStack` 类:

- `MinStack()` 初始化堆栈对象。
- `void push(int val)` 将元素val推入堆栈。
- `void pop()` 删除堆栈顶部的元素。
- `int top()` 获取堆栈顶部的元素。
- `int getMin()` 获取堆栈中的最小元素。

```python
class MinStack:
    def __init__(self):
        self.main_stack = []
        self.min_stack = []
    def push(self, val: int) -> None:
        self.main_stack.append(val)
        if not self.min_stack:
            self.min_stack.append(val)
        else:
            self.min_stack.append(min(val, self.min_stack[-1]))
    def pop(self) -> None:
        if self.main_stack:
            self.main_stack.pop()
            self.min_stack.pop()
    def top(self) -> int:
        if self.main_stack:
            return self.main_stack[-1]

    def getMin(self) -> int:
        if self.min_stack:
            return self.min_stack[-1]
# 时间复杂度：O(1)
# 空间复杂度：O(n)
```

```python
class Solution:
    def decodeString(self, s: str) -> str:
        stack = []
        current_num = 0 
        current_str = ""
        for char in s:
            if char.isdigit():
                current_num = current_num * 10 + int(char)
            elif char == '[':
                stack.append((current_str, current_num))
                current_str = ""
                current_num = 0
            elif char == ']':
                prev_str, num = stack.pop()
                current_str = prev_str + current_str * num
            else:
                current_str += char
        return current_str
# 时间复杂度：O(n+S)
```

```python
                answer[n] = i - n
            stack.append(i)
```

```python
class Solution:
    def jump(self, nums: List[int]) -> int:
        n = len(nums)
        maxpods, end, step = 0, 0, 0
        for i in range(n - 1):
            if maxpods >= i:
                maxpods = max(maxpods, i + nums[i])
                if i == end:
```

```python
            return 1
        if n == 2:
            return 2
        prev = 2
        prev_p = 1
        for i in range(3, n + 1):
            current = prev + prev_p
            prev_p = prev
            prev = current
        return prev
```

#### 279.完全平方数

题目：给你一个整数 `n` ，返回 *和为 `n` 的完全平方数的最少数量* 。**完全平方数** 是一个整数，其值等于另一个整数的平方；换句话说，其值等于一个整数自乘的积。例如，`1`、`4`、`9` 和 `16` 都是完全平方数，而 `3` 和 `11` 不是。

思路：从 1 算到 n，每个数都用所有比它小的平方数去凑，取最少个数

```python
class Solution:
    def numSquares(self, n: int) -> int:
        # dp[i] = 组成数字 i 所需的**最少平方数个数**
        dp = [float('inf')] * (n + 1)
        dp[0] = 0
        for i in range(1, n + 1):
             # 找出所有 ≤i 的平方数 j²
            max_j = int(math.sqrt(i))
            for j in range(1, max_j + 1):
                squre =  j * j
                # 核心：选这个平方数，总数 = dp[i - square] + 1
                dp[i] = min(dp[i], dp[i - squre] + 1)
        return dp[n]
```

题目：给你一个字符串 `s` 和一个字符串列表 `wordDict` 作为字典。如果可以利用字典中出现的一个或多个单词拼接出 `s` 则返回 `true`。

思路：dp [i] 表示字符串的前 i 个字符能不能被成功拆分

```python
class Solution:
    def wordBreak(self, s: str, wordDict: List[str]) -> bool:
        word_set = set(wordDict)
        n = len(s)
        dp = [False] * (n + 1)
        dp[0] = True
        for i in range(1, n + 1):
            for j in range(i):
                if dp[j] and s[j:i] in word_set:
                    dp[i] = True
                    break
        return dp[n]
# 时间复杂度：O(n^2)
# 空间复杂度：O(n)
```



#### 300.最长递增子系列

题目：给你一个整数数组 `nums` ，找到其中最长严格递增子序列的长度。**子序列** 是由数组派生而来的序列，删除（或不删除）数组中的元素而不改变其余元素的顺序。例如，`[3,6,2,7]` 是数组 `[0,3,1,6,2,2,7]` 的子序列。

思路：遍历每个数，让它接在前面所有比它小的数后面，记录能形成的最长长度，最后取最大

```python
class Solution:
    def lengthOfLIS(self, nums: List[int]) -> int:
        if not nums:
            return 0
        dp = []
        for i in range(len(nums)):
            dp.append(1)
            for j in range(i):
                if nums[i] > nums[j]:
                    dp[i] = max(dp[i], dp[j] + 1)
        return max(dp)
```

时间复杂度：`O(n²)`

空间复杂度：`O(n)`

#### 152.乘积最大子数组

思路：负数会让最大变最小、最小变最大。遍历时每遇到负数就交换 `cur_max` 和 `cur_min`，然后更新当前最大/最小乘积

```python
class Solution:
    def maxProduct(self, nums: List[int]) -> int:
        cur_max = cur_min = res = nums[0]
        for num in nums[1:]:
            if num < 0:
                cur_max, cur_min = cur_min, cur_max
            cur_max = max(num, cur_max * num)
            cur_min = min(num, cur_min * num)
            res = max(res, cur_max)
        return res
```

#### 416.分割等和子集

思路：转化为"能否从数组中选出若干数，使其和为 `sum/2`"。`dp[j]` 表示能否凑出和 `j`，倒序遍历防止重复选取

```python
class Solution:
    def canPartition(self, nums: List[int]) -> bool:
        total = sum(nums)
        if total % 2 != 0:
            return False
        target = total // 2
        n = len(nums)
        dp = [False] * (target + 1)
        dp[0] = True
        for num in nums:
            for j in range(target, num - 1, -1):
                dp[j] = dp[j] or dp[j - num]
        return dp[target]
```

**时间复杂度：** O(n*target)

#### 62.不同路径

题目：一个机器人位于一个 `m x n` 网格的左上角 （起始点在下图中标记为 “Start” ）。机器人每次只能向下或者向右移动一步。机器人试图达到网格的右下角（在下图中标记为 “Finish” ）。问总共有多少条不同的路径？

```python
class Solution:
    def uniquePaths(self, m: int, n: int) -> int:
        dp = [[1] * n for _ in range(m)]
        for i in range(1, m):
            for j in range(1, n):
                dp[i][j] = dp[i - 1][j] + dp[i][j - 1]
        return dp[m - 1][n - 1]
```

时间复杂度：`O(mn)`

空间复杂度：`O(min(m,n))`

#### 64.最小路径和

题目：给定一个包含非负整数的 `*m* x *n*` 网格 `grid` ，请找出一条从左上角到右下角的路径，使得路径上的数字总和为最小。**说明：**每次只能向下或者向右移动一步。

```python
class Solution:
    def minPathSum(self, grid: List[List[int]]) -> int:
        if not grid or not grid[0]:
            return 0
        rows = len(grid)
        columns = len(grid[0])
        dp = [[0] * columns for _ in range(rows)]
        dp[0][0] = grid[0][0]
        for i in range(1, rows):
            dp[i][0] = dp[i - 1][0] + grid[i][0]
        for j in range(1, columns):
            dp[0][j] = dp[0][j - 1] + grid[0][j]
        for i in range(1, rows):
            for j in range(1, columns):
                dp[i][j] = min(dp[i - 1][j], dp[i][j - 1]) + grid[i][j]
        return dp[rows - 1][columns - 1]
```

时间复杂度：`O(mn)`

空间复杂度：`O(mn)`

#### 5.最长回文子串

思路：回文串有奇偶两种中心（单字符/双字符）。从每个中心向两边扩展，记录最长回文串

```python
class Solution:
    def longestPalindrome(self, s: str) -> str:
        if not s or len(s) < 2:
            return s
        start, end = 0, 0
        def expandfuntion(left:int, right:int) -> int:
            while left >= 0 and right < len(s) and s[left] == s[right]:
                left -= 1
                right += 1
            return right - left  - 1
        for i in range(len(s)):
            len1 = expandfuntion(i, i)
            len2 = expandfuntion(i, i + 1)
            max_len = max(len1, len2)
            if max_len > end - start:
                start = i - (max_len-1) // 2
                end = i + max_len // 2
        return s[start:end + 1]
```

**时间复杂度：** O(n^2)

#### 1143.最长公共子系列

思路：`dp[i][j]` 表示 `text1[0..i-1]` 和 `text2[0..j-1]` 的 LCS 长度。字符相等则 `+1`，否则取左/上的最大值

```python
class Solution:
    def longestCommonSubsequence(self, text1: str, text2: str) -> int:
        m, n = len(text1), len(text2)
        dp = [[0] * (n + 1) for _ in range(m + 1)] 
        for i in range(1, m + 1):
            for j in range(1, n + 1):
                if text1[i - 1] == text2[j - 1]:
                    dp[i][j] = dp[i - 1][j - 1] + 1
                else:
                    dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])
        return dp[m][n]
```

时间复杂度是 O(m*n)

#### 72.编辑距离

题目：给你两个单词 `word1` 和 `word2`， 请返回将 `word1` 转换成 `word2` 所使用的最少操作数 。

你可以对一个单词进行如下三种操作：

- 插入一个字符
- 删除一个字符
- 替换一个字符

思路：`dp[i][j]` 表示：**把 `word1` 的前 `i` 个字符，变成 `word2` 的前 `j` 个字符，所需要的最少操作次数。**字符相等则不操作，否则取 增/删/改 的最小值 `+1`。初始化第一行/列为对应长度（全插入/删除）

```python
class Solution:
    def minDistance(self, word1: str, word2: str) -> int:
        m, n = len(word1), len(word2)
        dp = [[0] * (n + 1) for _ in range(m + 1)]
        for i in range(1, m + 1):
            dp[i][0] = i
        for j in range(1, n + 1):
            dp[0][j] = j
        for i in range(1, m + 1):
            for j in range(1, n + 1):
                # DP 表格下标 和 字符串下标 错开了一位
                if word1[i - 1] == word2[j - 1]:
                    dp[i][j] = dp[i - 1][j - 1]
                else:
                    dp[i][j] = min(dp[i-1][j-1], dp[i][j-1], dp[i-1][j]) + 1 # 替换、插入、删除
        return dp[m][n]
```

时间复杂度和空间复杂度都是 O(m*n)

#### 136.只出现一次的数字

题目：给你一个 **非空** 整数数组 `nums` ，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

思路：设计并实现线性时间复杂度的算法来解决此问题，且该算法只使用常量额外空间，使用位运算（异或），异或三个性质为任何数和 0 做异或运算，结果仍然是原来的数；任何数和其自身做异或运算，结果是 0；异或运算满足交换律和结合律。`reduce` 是 Python 高阶函数，作用是**对序列元素依次累积执行函数**

官解给的其他思路：

- 使用集合存储数字。遍历数组中的每个数字，如果集合中没有该数字，则将该数字加入集合，如果集合中已经有该数字，则将该数字从集合中删除，最后剩下的数字就是只出现一次的数字。
- 使用哈希表存储每个数字和该数字出现的次数。遍历数组即可得到每个数字出现的次数，并更新哈希表，最后遍历哈希表，得到只出现一次的数字。
- 使用集合存储数组中出现的所有数字，并计算数组中的元素之和。由于集合保证元素无重复，因此计算集合中的所有元素之和的两倍，即为每个元素出现两次的情况下的元素之和。由于数组中只有一个元素出现一次，其余元素都出现两次，因此用集合中的元素之和的两倍减去数组中的元素之和，剩下的数就是数组中只出现一次的数字。

```python
class Solution:
    def singleNumber(self, nums: List[int]) -> int:
        return reduce(lambda x, y: x ^ y, nums)
```

时间复杂度：`O(n)`

空间复杂度：`O(1)`

#### 75.颜色分类

题目：给定一个包含红色、白色和蓝色、共 `n` 个元素的数组 `nums` ，原地对它们进行排序，使得相同颜色的元素相邻，并按照红色、白色、蓝色顺序排列。我们使用整数 `0`、 `1` 和 `2` 分别表示红色、白色和蓝色。必须在不使用库内置的 sort 函数的情况下解决这个问题。

思路：一次遍历，用两个指针分别标记 0 和 1 的边界，边遍历边交换

```python
class Solution:
    def sortColors(self, nums: List[int]) -> None:
        n = len(nums)
        p0 = p1 = 0
        # p0：下一个 0 应该放置的位置（所有 < p0 的数都是 0）
        # p1：下一个 1 应该放置的位置（所有 < p1 的数都是 0 和 1）
        for i in range(n):
            if nums[i] == 1:
                nums[i], nums[p1] = nums[p1], nums[i]
                p1 += 1
            elif nums[i] == 0:
                nums[i], nums[p0] = nums[p0], nums[i]
                if p0 < p1:
                    nums[i], nums[p1] = nums[p1], nums[i]
                p0 += 1
                p1 += 1
```

时间复杂度：`O(n)`

空间复杂度：`O(1)`

#### 31.下一个排列

题目：整数数组的一个 **排列** 就是将其所有成员以序列或线性顺序排列。例如，`arr = [1,2,3]` ，以下这些都可以视作 `arr` 的排列：`[1,2,3]`、`[1,3,2]`、`[3,1,2]`、`[2,3,1]` 。

思路：找到比当前数字大的最小排列；如果已是最大排列，就返回最小排列（升序）

```python
class Solution:
    def nextPermutation(self, nums: List[int]) -> None:
        i = len(nums) - 2
        while i >= 0 and nums[i] >= nums[i + 1]:
            i -= 1
        if i >= 0:
            j = len(nums) - 1
            while j >= 0 and nums[i] >= nums[j]:
                j -= 1
            nums[i], nums[j] = nums[j], nums[i]
        left, right = i + 1, len(nums) - 1
        while left < right:
            nums[left], nums[right] = nums[right], nums[left]
            left += 1
            right -= 1
```

时间复杂度：`O(n)`

空间复杂度：`O(1)`

#### 287.寻找重复数

题目：给定一个包含 `n + 1` 个整数的数组 `nums` ，其数字都在 `[1, n]` 范围内（包括 `1` 和 `n`），可知至少存在一个重复的整数。假设 `nums` 只有 **一个重复的整数** ，返回 **这个重复的数** 。你设计的解决方案必须 **不修改** 数组 `nums` 且只用常量级 `O(1)` 的额外空间。

思路：把数组当成链表，找环的入口（入口值就是重复数）

```python
class Solution:
    def findDuplicate(self, nums: List[int]) -> int:
        slow = fast = 0
        while True:
            slow = nums[slow]
            fast = nums[nums[fast]]
            if slow == fast:
                break
        slow = 0
        while slow != fast:
            slow = nums[slow]
            fast = nums[fast]
        return slow
```

时间复杂度：`O(n)`

空间复杂度：`O(1)`
