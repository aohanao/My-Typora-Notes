# 力扣算法通用解法

## 1 哈希

#### 1.两数之和

思路：遍历数组时，用哈希表存储 `{值: 索引}`。对于当前数 `num`，检查 `target - num` 是否在表中，若存在则找到答案

```python
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        hashtable = dict()
        for i, num in enumerate(nums):
            if target - num in hashtable:
                return [hashtable[target - num], i]
            hashtable[nums[i]] = i 
        return []
```

时间复杂度：`O(n)`，其中 `n` 是数组中的元素数量

#### 49.字母异位词分组

思路：异位词排序后结果相同（如 `"eat"` 和 `"tea"` 排序后都是 `"aet"`）。用排序后的字符串作为哈希表的 key，将原字符串分组

```python
class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        mp = collections.defaultdict(list)
        for st in strs:
            key = "".join(sorted(st))
            mp[key].append(st)
        return list(mp.values())
```

时间复杂度：`O(nklogk)`，其中 `n` 是 `strs` 中的字符串的数量，`k` 是 `strs` 中的字符串的的最大长度。需要遍历 `n` 个字符串，对于每个字符串，需要 `O(klogk)` 的时间进行排序以及 `O(1)` 的时间更新哈希表，因此总时间复杂度是 `O(nklogk)`

#### 128.最长连续序列

思路：将所有数存入集合，只从"连续序列起点"（即 `num-1` 不在集合中）开始向右遍历，计算序列长度

```python
class Solution:
    def longestConsecutive(self, nums: List[int]) -> int:
        longest_streak = 0
        num_set = set(nums)
        for num in num_set:
            if num - 1 not in num_set:
                current_num = num
                current_streak = 1
                while current_num + 1 in num_set:
                    current_num += 1
                    current_streak += 1
                longest_streak = max(longest_streak, current_streak)
        return longest_streak
```

时间复杂度：`O(n)`，其中 `n`为数组的长度

## 2 双指针

#### 283.移动零

思路：`left` 指向待填充位置，`right` 遍历数组。遇到非零数就交换到 `left` 位置，`left` 前进。保证 `[0, left)` 都是非零数

```python
class Solution:
    def moveZeroes(self, nums: List[int]) -> None:
        n = len(nums)
        left = right = 0
        while right < n:
            if nums[right] != 0:
                nums[left], nums[right] =  nums[right], nums[left]
                left += 1
            right += 1
```

时间复杂度：`O(n)`

#### 11.盛最多水的容器

思路：面积 = 底 × 高。底边长度固定时，移动较短的那根板（因为移动长的板，底边变小、高度不会增加，面积必然变小）

```python
class Solution:
    def maxArea(self, height: List[int]) -> int:
        n = len(height)
        left = 0
        right = n - 1
        ml = 0
        maxml = 0
        while left < right:
            ml = (right - left) * min(height[left], height[right])
            maxml =  max(maxml, ml)
            if height[left] < height[right]:
                left += 1
            else:
                right -= 1
        return maxml
```

时间复杂度：`O(n)`

#### 15.三数之和

思路：先排序，固定第一个数，后两个数用双指针从两端向中间找。注意跳过重复元素避免重复解

```python
class Solution:
    def threeSum(self, nums: List[int]) -> List[List[int]]:
        n = len(nums)
        nums.sort()
        ans = list()
        for first in range(n):
            if first > 0 and nums[first] == nums[first - 1]:
                continue
            third = n - 1
            target = -nums[first]
            for second in range(first + 1, n):
                if second > first + 1 and nums[second] == nums[second - 1]:
                    continue
                while second < third and nums[second] + nums[third] > target:
                    third -= 1
                if second == third:
                    break
                if nums[second] + nums[third] == target:
                    ans.append([nums[first], nums[second], nums[third]])
        return ans
```

时间复杂度：`O(n^2)`

#### 43.接雨水

思路：哪边矮就先处理哪边（因为矮边决定水位）。用 `leftMax/rightMax` 记录两侧最大高度，当前位置能接的水 = `maxHeight - currentHeight`

```python
class Solution:
    def trap(self, height: List[int]) -> int:
        n = len(height)
        left, leftMax, rightMax = 0, 0, 0
        right = n - 1
        ans = 0
        while left < right:
            leftMax = max(leftMax, height[left])
            rightMax = max(rightMax, height[right])
            if height[left] < height[right]:
                ans += leftMax - height[left]
                left += 1
            else:
                ans += rightMax - height[right]
                right -= 1
        return ans
```

时间复杂度：`O(n)`，其中 `n` 是数组 `height` 的长度。两个指针的移动总次数不超过 `n`

## 3 滑动窗口

#### 3.无重复字符的最长子串

思路：右指针 `rk` 扩张窗口，用集合判断字符是否重复；遇到重复则左指针 `i` 右移收缩窗口。记录窗口最大长度

```python
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        occ = set()
        n = len(s)
        rk, ans = 0, 0
        for i in range(n):
            if i != 0:
                occ.remove(s[i - 1])
            while rk < n and s[rk] not in occ:
                occ.add(s[rk])
                rk += 1
            ans = max(ans, rk - i)
        return ans
```

时间复杂度：*O*(*N*)，其中 *N* 是字符串的长度。左指针和右指针分别会遍历整个字符串一次。

#### 438.找到字符串中所有字母异位词

思路：窗口长度固定为 `p` 的长度，用哈希表统计字符频次。滑动时更新频次表，与 `p` 的频次表对比判断是否匹配

```python
from typing import List

class Solution:
    def findAnagrams(self, s: str, p: str) -> List[int]:
        s_len, p_len = len(s), len(p)
        if s_len < p_len:
            return []
        
        # 统计p的字符频次（字典）
        p_count = {}
        for char in p:
            p_count[char] = p_count.get(char, 0) + 1
        
        # 初始化窗口内s的字符频次（字典）
        s_count = {}
        for i in range(p_len):
            char = s[i]
            s_count[char] = s_count.get(char, 0) + 1
        
        ans = []
        # 初始窗口匹配检查
        if s_count == p_count:
            ans.append(0)
        
        # 滑动窗口更新
        for i in range(s_len - p_len):
            # 移出窗口左侧字符
            left_char = s[i]
            s_count[left_char] -= 1
            if s_count[left_char] == 0:  # 频次为0时删除键，避免干扰对比
                del s_count[left_char]
            
            # 移入窗口右侧新字符
            right_char = s[i + p_len]
            s_count[right_char] = s_count.get(right_char, 0) + 1
            
            # 对比当前窗口与p的频次
            if s_count == p_count:
                ans.append(i + 1)
        
        return ans
```

时间复杂度：O(m+(n−m)×Σ)，其中 n 为字符串 s 的长度，m 为字符串 p 的长度，Σ 为所有可能的字符数。我们需要 O(m) 来统计字符串 p 中每种字母的数量；需要 O(m) 来初始化滑动窗口；需要判断 n−m+1 个滑动窗口中每种字母的数量是否与字符串 p 中每种字母的数量相同，每次判断需要 O(Σ) 。因为 s 和 p 仅包含小写字母，所以 Σ=26。

## 4 子串

#### 560.和为k的子数组

思路：`前缀和 s[i]` 表示 `nums[0..i]` 的和。若 `s[j] - s[i] = k`，则 `nums[i+1..j]` 和为 k。遍历时用哈希表记录前缀和出现次数，`ans += count[s - k]`

```python
class Solution:
    def subarraySum(self, nums: List[int], k: int) -> int:
        mp = defaultdict(int) 
        s = ans = 0
        for num in nums:
            mp[s] += 1
            s += num
            ans += mp[s - k]
        return ans
```

时间复杂度：`O(n)`

## 5 普通数组

#### 53.最大子数组和

思路：`current` 维护以当前元素结尾的最大子数组和。若 `current + x < x`，则丢弃前面的子数组，从 `x` 重新开始。`maxSum` 记录全局最大

```python
class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        # 初始化当前子数组和current、全局最大和maxSum为数组第一个元素
        # 因为子数组至少包含一个元素，初始值不能设为0（避免全负数的情况）
        current = maxSum = nums[0]
        # 从数组第二个元素开始遍历（第一个元素已作为初始值）
        for x in nums[1:]:
            # 情况1：将当前元素x加入前一个子数组，和为current + x
            # 情况2：以当前元素x为起点，重新开始一个子数组，和为x
            # 取两种情况的较大值作为新的当前子数组和
            current = max(x, current + x)
            # 更新全局最大和：比较当前子数组和与历史最大和，保留更大值
            maxSum = max(current, maxSum)
        return maxSum
```

#### 56.合并区间

按左端点升序，比较当前数组右端点与下一数组左端点，合并后存入新数组

```python
class Solution:
    def merge(self, intervals: List[List[int]]) -> List[List[int]]:
        # 先按照每个区间的起始位置进行排序
        intervals.sort(key=lambda x: x[0])
        merged = []  # 用于存放合并后的区间
        for interval in intervals:
            # 如果 merged 为空，或者当前区间和 merged 最后一个区间不重叠，则直接加入
            # 不重叠的判断条件：上一区间的结束 < 当前区间的开始
            if not merged or merged[-1][1] < interval[0]:
                merged.append(interval)
            else:
                # 否则，说明有重叠，合并区间：
                # 更新 merged 最后一个区间的结束位置为两者中较大的那个
                merged[-1][1] = max(merged[-1][1], interval[1])
        return merged
```

排序为 Python 内置的 `sort()` 方法，底层实现是 **Timsort** 算法，属于高效的比较类排序，时间复杂度固定为 **O(n log n)**（`n` 是区间数量）

#### 189.轮转数组

思路：通过三次反转数组，实现数组向右旋转 k 位

```python
class Solution:
    def rotate(self, nums: List[int], k: int) -> None:
        n = len(nums)
        k %= n
        def reverse(start, end):
            while start < end:
                nums[start], nums[end] = nums[end], nums[start]
                start += 1
                end -= 1
        reverse(0, n - 1)
        reverse(0, k - 1)
        reverse(k, n - 1)
```

时间复杂度为: `O(n)`

#### 238.除了自身以外数组的乘积

思路：双指针同步遍历数组，用两个变量分别累积左右乘积，一次遍历直接算出每个位置「左侧乘积 × 右侧乘积」的结果

```python
class Solution:
    def productExceptSelf(self, nums: List[int]) -> List[int]:
        n = len(nums)
        left = 0
        right = n - 1
        answer = [1] * n
        i = j = 1
        while left < n:
            answer[left] *= i
            i *= nums[left]
            answer[right] *= j
            j *= nums[right]
            left += 1
            right -= 1
        return answer
```

时间复杂度为: `O(n)`

## 6 矩阵

#### 73.矩阵置零

思路：先用两个数组标记出**所有出现过 0 的行和列**，再根据这些标记，把这些行和列上的所有数字全部变成 0

```python
class Solution:
    def setZeroes(self, matrix: List[List[int]]) -> None:
        m = len(matrix)
        n = len(matrix[0])
        row = [False] * m
        col = [False] * n
        for i in range(m):
            for j in range(n):
                if matrix[i][j] == 0:
                    row[i] = col[j] = True
        for i in range(m):
            for j in range(n):
                if row[i] or col[j]:
                    matrix[i][j] = 0
```

时间复杂度：O (m × n)，m 是矩阵行数，n 是矩阵列数

#### 54.螺旋矩阵

```python
class Solution:
    def spiralOrder(self, matrix: List[List[int]]) -> List[int]:
        m, n = len(matrix), len(matrix[0])
        top, bottom = 0, m - 1
        left, right = 0, n - 1

        res = []
        direction = 0  # 0:右, 1:下, 2:左, 3:上

        while left <= right and top <= bottom:

            if direction == 0:  # 向右
                for col in range(left, right + 1):
                    res.append(matrix[top][col])
                top += 1

            elif direction == 1:  # 向下
                for row in range(top, bottom + 1):
                    res.append(matrix[row][right])
                right -= 1

            elif direction == 2:  # 向左
                for col in range(right, left - 1, -1):
                    res.append(matrix[bottom][col])
                bottom -= 1

            else:  # direction == 3，向上
                for row in range(bottom, top - 1, -1):
                    res.append(matrix[row][left])
                left += 1

            direction = (direction + 1) % 4  # 方向切换

        return res
```

class Solution:
    def searchMatrix(self, matrix: List[List[int]], target: int) -> bool:
        m, n = len(matrix), len(matrix[0])
        left, right = 0, n - 1
        while left < m and right >= 0:

            if matrix[left][right] > target:
                right -= 1
            else:
                left += 1
        return False

#### 200.岛屿数量

题目：给你一个由 `'1'`（陆地）和 `'0'`（水）组成的的二维网格，请你计算网格中岛屿的数量。岛屿总是被水包围，并且每座岛屿只能由水平方向和/或竖直方向上相邻的陆地连接形成。此外，你可以假设该网格的四条边均被水包围。

思路：采用深度优先搜索（DFS），遍历二维网格，遇到未访问的陆地`1`时，岛屿数量加 1，并通过 DFS 将其上下左右所有相连的陆地全部标记为水域`0`（避免重复统计），最终统计出总岛屿数。

```python
class Solution:
    def dfs(self, grid, r, c):
        grid[r][c] = 0
        nr = len(grid)
        nc = len(grid[0])
        for x, y in [(r - 1, c), (r + 1, c), (r, c - 1), (r, c + 1)]:
            if 0 <= x < nr and 0 <= y < nc and grid[x][y] == "1":
                self.dfs(grid, x, y)
    def numIslands(self, grid: List[List[str]]) -> int:
        nr = len(grid)
        if nr == 0:
            return 0
        nc = len(grid[0])
        num_islands = 0
        for r in range(nr):
            for c in range(nc):
                if grid[r][c] == "1":
                    num_islands += 1
                    self.dfs(grid, r, c)
        return nmu_islands
```

时间复杂度：**O(m×n)**（m 为行数，n 为列数，每个格子仅访问一次）

空间复杂度：**最坏 O (m×n)**（递归栈深度，全为陆地时），平均**O(min(m,n))**

#### 994.腐烂的橘子

题目：在给定的 `m x n` 网格 `grid` 中，每个单元格可以有以下三个值之一：

- 值 `0` 代表空单元格；
- 值 `1` 代表新鲜橘子；
- 值 `2` 代表腐烂的橘子。

每分钟，腐烂的橘子 **周围 4 个方向上相邻** 的新鲜橘子都会腐烂。

返回 直到单元格中没有新鲜橘子为止所必须经过的最小分钟数。如果不可能，返回 `-1` 。

思路：采用**BFS 广度优先搜索**（多源队列），先收集所有初始腐烂橘子入队，统计新鲜橘子数量；每一轮遍历队列中所有当前腐烂橘子，向四向扩散感染新鲜橘子，将其变为腐烂并加入队列，每轮记一分钟；直到无新鲜橘子或无腐烂橘子可扩散，最终新鲜橘子全烂则返回时间，否则返回 - 1

```python
class Solution:
    def orangesRotting(self, grid: List[List[int]]) -> int:
        rows, cols = len(grid), len(grid[0])
        # 初始化队列，用于BFS广度优先搜索（存储腐烂橘子的坐标）
        queue = collections.deque()
        fresh_count = 0
        for r in range(rows):
            for c in range(cols):
                if grid[r][c] == 2:
                    queue.append((r, c))
                elif grid[r][c] == 1:
                    fresh_count += 1
        if fresh_count == 0:
            return 0
        minutes = 0
        # BFS循环：队列不为空 且 还有新鲜橘子
        while queue and fresh_count > 0:
            minutes += 1
            # 一次性处理当前队列里所有腐烂橘子（一层）
            for _ in range(len(queue)):
                r, c = queue.popleft()
                for dr, dc in [(1,0), (-1,0), (0,1), (0,-1)]:
                    nr, nc = r + dr, c + dc
                    if 0 <= nr < rows and 0 <= nc < cols and grid[nr][nc] == 1:
                        grid[nr][nc] = 2
                        fresh_count -= 1
                        queue.append((nr, nc))
        return minutes if fresh_count == 0 else -1
```

时间复杂度：O(nm)。即进行一次广度优先搜索的时间，其中 n,m 分别为 grid 的行数与列数。

空间复杂度：O(nm)。需要额外的 dis 数组记录每个新鲜橘子被腐烂的最短时间，大小为 O(nm)，且广度优先搜索中队列里存放的状态最多不会超过 nm 个，最多需要 O(nm) 的空间，所以最后的空间复杂度为 O(nm)。

#### 207.课程表

题目：你这个学期必须选修 `numCourses` 门课程，记为 `0` 到 `numCourses - 1` 。

在选修某些课程之前需要一些先修课程。 先修课程按数组 `prerequisites` 给出，其中 `prerequisites[i] = [ai, bi]` ，表示如果要学习课程 `ai` 则 **必须** 先学习课程 `bi` 。

- 例如，先修课程对 `[0, 1]` 表示：想要学习课程 `0` ，你需要先完成课程 `1` 。

请你判断是否可能完成所有课程的学习？如果可以，返回 `true` ；否则，返回 `false` 。

思路：这是**拓扑排序（BFS 版）**，用于判断课程依赖图是否**有环**（无环就能上完所有课）。

1. 先构建课程有向图，并统计每门课的**入度（前置课程数）**。
2. 把**入度为 0**（无前置课）的课程加入队列，开始学习。
3. 每学完一门课，就减少它后续课程的入度；当某门课入度变为 0，就加入队列学习。
4. 最终**能学完的课程数 == 总课程数**，说明无环，返回 True；否则有环，返回 False。

```python
class Solution:
    def canFinish(self, numCourses: int, prerequisites: List[List[int]]) -> bool:
        # 邻接表：存储课程依赖图
        graph = defaultdict(list)
        # 入度数组：记录每门课需要的前置课程数量
        indegree = [0] * numCourses
        # 构建图 + 统计入度
        # dest: 目标课, src: 前置课 (先学src才能学dest)
        for dest, src in prerequisites:
            graph[src].append(dest)
            indegree[dest] += 1
        # 初始化队列：所有入度为0（无前置课）的课程
        queue = deque([i for i in range(numCourses) if indegree[i] == 0])
        learned_course = 0
        # BFS拓扑排序
        while queue:
            course = queue.popleft()
            learned_course += 1
            for next_course in graph[course]:
                indegree[next_course] -= 1
                if indegree[next_course] == 0:
                    queue.append(next_course)
        # 能学完所有课 = 无环 = 返回Tru
        return learned_course == numCourses
```

时间复杂度: *O*(*n*+*m*)，其中 *n* 为课程数，*m* 为先修课程的要求数。这其实就是对图进行深度优先搜索的时间复杂度。

空间复杂度: *O*(*n*+*m*)

#### 208.实现Trie（前缀树）

题目：**[Trie](https://baike.baidu.com/item/字典树/9825209?fr=aladdin)**（发音类似 "try"）或者说 **前缀树** 是一种树形数据结构，用于高效地存储和检索字符串数据集中的键。这一数据结构有相当多的应用情景，例如自动补全和拼写检查。

请你实现 Trie 类：

- `Trie()` 初始化前缀树对象。
- `void insert(String word)` 向前缀树中插入字符串 `word` 。
- `boolean search(String word)` 如果字符串 `word` 在前缀树中，返回 `true`（即，在检索之前已经插入）；否则，返回 `false` 。
- `boolean startsWith(String prefix)` 如果之前已经插入的字符串 `word` 的前缀之一为 `prefix` ，返回 `true` ；否则，返回 `false` 。

思路：这是字典树（Trie）的标准实现，专门高效存储和查询字符串：

1. 每个节点存 26 个字母子节点 + 标记是否为单词结尾。
2. **insert**：逐字符遍历，不存在则新建节点，走到最后标记单词结束。
3. **search**：按字符走到末尾，若节点存在且标记结束则返回 True。
4. **startsWith**：只需按前缀能走到末尾就返回 True，不要求是完整单词。

```python
class Trie:

    def __init__(self):
        self.children = [None] * 26
        self.isEnd = False

    def searchPrefix(self, prefix: str) -> "Trie":
        node = self
        for ch in prefix:
            ch = ord(ch) - ord("a")
            if not node.children[ch]:
                return None
            node = node.children[ch]
        return node

    def insert(self, word: str) -> None:
        node = self
        for ch in word:
            ch =ord(ch) - ord("a")
            if not node.children[ch]:
                node.children[ch] = Trie()
            node = node.children[ch]
        node.isEnd = True

    def search(self, word: str) -> bool:
        node = self.searchPrefix(word)
        return node is not None and node.isEnd

    def startsWith(self, prefix: str) -> bool:
        return self.searchPrefix(prefix) is not None
```

时间复杂度：插入 / 搜索 / 前缀查询均为 **O(L)**，L 是字符串长度。

空间复杂度：**O(26×L)**，L 为所有存入字符串总字符数。

## 10 回溯

回溯的本质是：**做选择 → 递归探索 → 撤销选择**，核心是通过「试错」的方式遍历所有可能的解空间。

我先给你一个通用的伪代码模板，再拆解每个部分的作用：

```python
def backtrack(参数):
    # 1. 终止条件：找到一个合法解，记录结果
    if 满足终止条件:
        把当前路径加入结果集
        return
    
    # 2. 遍历所有可选的「选择」
    for 选择 in 选择列表:
        # 剪枝：排除不合法/无效的选择（可选，但能大幅提升效率）
        if 选择不合法:
            continue/break
        
        # 3. 做出选择：把当前选择加入路径
        路径.append(选择)
        
        # 4. 递归探索：进入下一层决策树
        backtrack(更新后的参数)
        
        # 5. 撤销选择：回溯，回到上一步（关键！）
        路径.pop()

# 主函数：初始化结果集和路径，调用回溯函数
def main(问题参数):
    结果集 = []
    路径 = []
    backtrack(初始参数)
    return 结果集
```

### 模板核心拆解（5 个关键部分）

我用「人话 + 对应组合总和案例」帮你理解每个部分的作用：

| 模板部分 |              核心作用              |             组合总和中的对应实现             |
| :------: | :--------------------------------: | :------------------------------------------: |
| 终止条件 |  划定递归的边界，找到有效解时记录  | `if remain == 0: result.append(path.copy())` |
| 遍历选择 |      遍历当前层所有可做的选择      |   `for i in range(start, len(candidates))`   |
|   剪枝   |    提前排除无效选择，避免走弯路    |    `if num > remain: break`（排序后剪枝）    |
|  做选择  | 将当前选择加入「路径」（记录已选） |              `path.append(num)`              |
| 递归探索 |       带着新的状态进入下一层       |      `backtrack(remain - num, path, i)`      |
| 撤销选择 |      回到上一层，尝试其他选择      |                 `path.pop()`                 |

### 模板的灵活适配（不同题型的调整点）

你只需要根据题目的**核心约束**，调整模板中的「选择列表」和「参数」即可，我用常见题型举例：

#### 1. 组合类问题（如组合总和、组合总和 II、组合）

- **核心约束**：元素可重复选 / 不可重复选、组合不重复（[2,3] 和 [3,2] 算同一个）

- **调整点**：

  - 用 `start` 参数控制「选择列表的起始位置」（避免重复组合）；
  - 可重复选：`backtrack(..., i)`（下一层仍从当前索引开始）；
  - 不可重复选：`backtrack(..., i+1)`（下一层从下一个索引开始）。


#### 2. 排列类问题（如全排列、全排列 II）

- **核心约束**：元素可重复 / 不可重复、排列要去重

- **调整点**：

  - 不用 `start` 参数，而是用「used 数组」标记已选元素；
  - 去重：排序后跳过重复元素 + 检查 `used` 数组。


#### 3. 子集类问题（如子集、子集 II）

- **核心约束**：收集所有子集、子集不重复

- **调整点**：

  - 终止条件不是「凑够目标」，而是「遍历完所有元素」（或每一步都把路径加入结果）；
  - 用 `start` 参数控制起始位置，避免重复子集。


#### 4. 棋盘类问题（如 N 皇后、解数独）

- **核心约束**：满足棋盘规则（如皇后不冲突）
- **调整点**：
  - 「选择列表」是当前行的所有列；
  - 「剪枝」是检查当前位置是否符合规则（如皇后是否冲突）；
  - 终止条件是「遍历完所有行」。

#### 155.最小栈

```python
class MinStack:

    def __init__(self):
        self.main_stack = []
        self.min_stack = []

    def push(self, val: int) -> None:
        self.main_stack.append(val)
        if not self.min_stack:
            self.min_stack.append(val)
        else:
            self.min_stack.append(min(val, self.min_stack[-1]))

    def pop(self) -> None:
        if self.main_stack:
            self.main_stack.pop()
            self.min_stack.pop()

    def top(self) -> int:
        if self.main_stack:
            return self.main_stack[-1]

    def getMin(self) -> int:
        if self.min_stack:
            return self.min_stack[-1]
```

#### 394.字符串编码

思路：遇到 `[` 将当前字符串和重复次数压栈，遇到 `]` 弹出栈顶拼接（`prev_str + cur_str * num`）。数字可能多位，需累积计算

```python
class Solution:
    def decodeString(self, s: str) -> str:
        stack = []
        current_num = 0 
        current_str = ""
        for char in s:
            if char.isdigit():
                current_num = current_num * 10 + int(char)
            elif char == '[':
                stack.append((current_str, current_num))
                current_str = ""
                current_num = 0
            elif char == ']':
                prev_str, num = stack.pop()
                current_str = prev_str + current_str * num
            else:
                current_str += char
        return current_str
```

#### 739.每日温度

思路：栈存索引，保持温度递减。当前温度 > 栈顶温度时，说明找到了栈顶元素的"下一个更大元素"，计算天数差并出栈

```python
class Solution:
    def dailyTemperatures(self, temperatures: List[int]) -> List[int]:
        length = len(temperatures)
        stack = []
        answer = [0] * length
        for i in range(length):
            while stack and temperatures[i] > temperatures[stack[-1]]:
                n = stack.pop()
                answer[n] = i - n
            stack.append(i)
        return answer
```

class Solution:

            elif price - min_price > max_profit:
                max_profit = price - min_price
        return max_profit

#### 55.跳跃游戏

题目：给你一个非负整数数组 `nums` ，你最初位于数组的 **第一个下标** 。数组中的每个元素代表你在该位置可以跳跃的最大长度。判断你是否能够到达最后一个下标，如果可以，返回 `true` ；否则，返回 `false` 。

思路：从头到尾遍历，维护一个能跳到的最远距离，只要当前位置在范围内，就更新最远距离；如果最远距离能覆盖最后一个位置，就说明能跳到终点。

```python
class Solution:
    def canJump(self, nums: List[int]) -> bool:
        n = len(nums)
        right = 0
        for i in range(n):
            if i <= right:
                right = max(right, i + nums[i])
                if right >= n - 1:
                    return True
        return False
```

时间复杂度：`O(n)`

空间复杂度：`O(1)`

#### 45.跳跃游戏II

题目：给定一个长度为 `n` 的 **0 索引**整数数组 `nums`。初始位置在下标 0。每个元素 `nums[i]` 表示从索引 `i` 向后跳转的最大长度。换句话说，如果你在索引 `i` 处，你可以跳转到任意 `(i + j)` 处：`0 <= j <= nums[i]` 且`i + j < n`，返回到达 `n - 1` 的最小跳跃次数。测试用例保证可以到达 `n - 1`。

思路：在当前能跳到的范围内，找到能跳得最远的位置；到达范围边界时，必须跳一步，把边界更新为最远位置。最终统计最少跳几次

```python
class Solution:
    def jump(self, nums: List[int]) -> int:
        n = len(nums)
        maxpods, end, step = 0, 0, 0
        for i in range(n - 1):
            if maxpods >= i:
                maxpods = max(maxpods, i + nums[i])
                if i == end:
                    end = maxpods
                    step += 1
        return step
```

时间复杂度：`O(n)`

空间复杂度：`O(1)`

#### 763.划分字母区间

题目：给你一个字符串 `s` 。我们要把这个字符串划分为尽可能多的片段，同一字母最多出现在一个片段中。例如，字符串 `"ababcc"` 能够被分为 `["abab", "cc"]`，但类似 `["aba", "bcc"]` 或 `["ab", "ab", "cc"]` 的划分是非法的。注意，划分结果需要满足：将所有划分结果按顺序连接，得到的字符串仍然是 `s` 。返回一个表示每个字符串片段的长度的列表。

思路：先记录每个字母最后一次出现的位置，从头遍历，不断扩大当前片段的结束位置，当遍历到结束位置时，分割一次，记录长度，最终返回所有分割后的长度

```python
class Solution:
    def partitionLabels(self, s: str) -> List[int]:
        # 1. 先存：每个字母【最后一次出现的下标】
        last = [0] * 26
        for i, ch in enumerate(s):
            # 把字母转成 0~25 的数字
            last[ord(ch) - ord("a")] = i
        partition = list()
        start = end = 0
        for i, ch in enumerate(s):
            # 关键：当前片段的结束位置 = 能覆盖到的最远的最后出现位置
            end = max(end, last[ord(ch) - ord("a")])
            if i == end:
                partition.append(end - start + 1)
                start = end + 1
        return partition
```

时间复杂度：`O(n)`

#### 152.乘积最大子数组

思路：负数会让最大变最小、最小变最大。遍历时每遇到负数就交换 `cur_max` 和 `cur_min`，然后更新当前最大/最小乘积

```python
class Solution:
    def maxProduct(self, nums: List[int]) -> int:
        cur_max = cur_min = res = nums[0]
        for num in nums[1:]:
            if num < 0:
                cur_max, cur_min = cur_min, cur_max
            cur_max = max(num, cur_max * num)
            cur_min = min(num, cur_min * num)
            res = max(res, cur_max)
        return res
```

#### 416.分割等和子集

思路：转化为"能否从数组中选出若干数，使其和为 `sum/2`"。`dp[j]` 表示能否凑出和 `j`，倒序遍历防止重复选取

```python
class Solution:
    def canPartition(self, nums: List[int]) -> bool:
        total = sum(nums)
        if total % 2 != 0:
            return False
        target = total // 2
        n = len(nums)
        dp = [False] * (target + 1)
        dp[0] = True
        for num in nums:
            for j in range(target, num - 1, -1):
                dp[j] = dp[j] or dp[j - num]
        return dp[target]
```

```python
class Solution:
    def uniquePaths(self, m: int, n: int) -> int:
        dp = [[1] * n for _ in range(m)]
        for i in range(1, m):
            for j in range(1, n):
                dp[i][j] = dp[i - 1][j] + dp[i][j - 1]
        return dp[m - 1][n - 1]
```

#### 64.最小路径和

题目：给定一个包含非负整数的 `*m* x *n*` 网格 `grid` ，请找出一条从左上角到右下角的路径，使得路径上的数字总和为最小。**说明：**每次只能向下或者向右移动一步。

```python
class Solution:
    def minPathSum(self, grid: List[List[int]]) -> int:
        if not grid or not grid[0]:
            return 0
        rows = len(grid)
        columns = len(grid[0])
        dp = [[0] * columns for _ in range(rows)]
        dp[0][0] = grid[0][0]
        for i in range(1, rows):
            dp[i][0] = dp[i - 1][0] + grid[i][0]
        for j in range(1, columns):
            dp[0][j] = dp[0][j - 1] + grid[0][j]
        for i in range(1, rows):
            for j in range(1, columns):
                dp[i][j] = min(dp[i - 1][j], dp[i][j - 1]) + grid[i][j]
        return dp[rows - 1][columns - 1]
```

时间复杂度：`O(mn)`

空间复杂度：`O(mn)`

#### 5.最长回文子串

思路：回文串有奇偶两种中心（单字符/双字符）。从每个中心向两边扩展，记录最长回文串

```python
class Solution:
    def longestPalindrome(self, s: str) -> str:
        if not s or len(s) < 2:
            return s
        start, end = 0, 0
        def expandfuntion(left:int, right:int) -> int:
            while left >= 0 and right < len(s) and s[left] == s[right]:
                left -= 1
                right += 1
            return right - left  - 1
        for i in range(len(s)):
            len1 = expandfuntion(i, i)
            len2 = expandfuntion(i, i + 1)
            max_len = max(len1, len2)
            if max_len > end - start:
                start = i - (max_len-1) // 2
                end = i + max_len // 2
        return s[start:end + 1]
```

**时间复杂度：** O(n^2)

#### 1143.最长公共子系列

思路：`dp[i][j]` 表示 `text1[0..i-1]` 和 `text2[0..j-1]` 的 LCS 长度。字符相等则 `+1`，否则取左/上的最大值

```python
class Solution:
    def longestCommonSubsequence(self, text1: str, text2: str) -> int:
        m, n = len(text1), len(text2)
        dp = [[0] * (n + 1) for _ in range(m + 1)] 
        for i in range(1, m + 1):
            for j in range(1, n + 1):
                if text1[i - 1] == text2[j - 1]:
                    dp[i][j] = dp[i - 1][j - 1] + 1
                else:
                    dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])
        return dp[m][n]
```

时间复杂度是 O(m*n)

#### 72.编辑距离

思路：`dp[i][j]` 表示 `word1[0..i-1]` 转成 `word2[0..j-1]` 的最小操作数。字符相等则不操作，否则取 增/删/改 的最小值 `+1`。初始化第一行/列为对应长度（全插入/删除）

```python
class Solution:
    def minDistance(self, word1: str, word2: str) -> int:
        m, n = len(word1), len(word2)
        dp = [[0] * (n + 1) for _ in range(m + 1)]
        for i in range(1, m + 1):
            dp[i][0] = i
        for j in range(1, n + 1):
            dp[0][j] = j
        for i in range(1, m + 1):
            for j in range(1, n + 1):
                if word1[i - 1] == word2[j - 1]:
                    dp[i][j] = dp[i - 1][j - 1]
                else:
                    dp[i][j] = min(dp[i-1][j-1], dp[i][j-1], dp[i-1][j]) + 1
        return dp[m][n]
```

时间复杂度和空间复杂度都是 O(m*n)

#### 136.只出现一次的数字

题目：给你一个 **非空** 整数数组 `nums` ，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

思路：设计并实现线性时间复杂度的算法来解决此问题，且该算法只使用常量额外空间，使用位运算（异或），异或三个性质为任何数和 0 做异或运算，结果仍然是原来的数；任何数和其自身做异或运算，结果是 0；异或运算满足交换律和结合律。`reduce` 是 Python 高阶函数，作用是**对序列元素依次累积执行函数**

官解给的其他思路：

- 使用集合存储数字。遍历数组中的每个数字，如果集合中没有该数字，则将该数字加入集合，如果集合中已经有该数字，则将该数字从集合中删除，最后剩下的数字就是只出现一次的数字。
- 使用哈希表存储每个数字和该数字出现的次数。遍历数组即可得到每个数字出现的次数，并更新哈希表，最后遍历哈希表，得到只出现一次的数字。
- 使用集合存储数组中出现的所有数字，并计算数组中的元素之和。由于集合保证元素无重复，因此计算集合中的所有元素之和的两倍，即为每个元素出现两次的情况下的元素之和。由于数组中只有一个元素出现一次，其余元素都出现两次，因此用集合中的元素之和的两倍减去数组中的元素之和，剩下的数就是数组中只出现一次的数字。

```python
class Solution:
    def singleNumber(self, nums: List[int]) -> int:
        return reduce(lambda x, y: x ^ y, nums)
```

时间复杂度：`O(n)`

空间复杂度：`O(1)`

#### 169.多数元素

题目：给定一个大小为 `n` 的数组 `nums` ，返回其中的多数元素。多数元素是指在数组中出现次数 **大于**  n/2  的元素。你可以假设数组是非空的，并且给定的数组总是存在多数元素。

思路：我们知道出现次数最多的元素大于 n/2 次，所以可以用哈希表来快速统计每个元素出现的次数

```python
class Solution:
    def majorityElement(self, nums: List[int]) -> int:
        counts = collections.Counter(nums)
        return max(counts.keys(), key = counts.get)
```

时间复杂度：`O(n)`

空间复杂度：`O(n)`

#### 75.颜色分类

题目：给定一个包含红色、白色和蓝色、共 `n` 个元素的数组 `nums` ，原地对它们进行排序，使得相同颜色的元素相邻，并按照红色、白色、蓝色顺序排列。我们使用整数 `0`、 `1` 和 `2` 分别表示红色、白色和蓝色。必须在不使用库内置的 sort 函数的情况下解决这个问题。

思路：一次遍历，用两个指针分别标记 0 和 1 的边界，边遍历边交换

```python
class Solution:
    def sortColors(self, nums: List[int]) -> None:
        n = len(nums)
        p0 = p1 = 0
        # p0：下一个 0 应该放置的位置（所有 < p0 的数都是 0）
        # p1：下一个 1 应该放置的位置（所有 < p1 的数都是 0 和 1）
        for i in range(n):
            if nums[i] == 1:
                nums[i], nums[p1] = nums[p1], nums[i]
                p1 += 1
            elif nums[i] == 0:
                nums[i], nums[p0] = nums[p0], nums[i]
                if p0 < p1:
                    nums[i], nums[p1] = nums[p1], nums[i]
                p0 += 1
                p1 += 1
```

时间复杂度：`O(n)`

空间复杂度：`O(1)`

#### 31.下一个排列

题目：整数数组的一个 **排列** 就是将其所有成员以序列或线性顺序排列。例如，`arr = [1,2,3]` ，以下这些都可以视作 `arr` 的排列：`[1,2,3]`、`[1,3,2]`、`[3,1,2]`、`[2,3,1]` 。

思路：找到比当前数字大的最小排列；如果已是最大排列，就返回最小排列（升序）

```python
class Solution:
    def nextPermutation(self, nums: List[int]) -> None:
        i = len(nums) - 2
        while i >= 0 and nums[i] >= nums[i + 1]:
            i -= 1
        if i >= 0:
            j = len(nums) - 1
            while j >= 0 and nums[i] >= nums[j]:
                j -= 1
            nums[i], nums[j] = nums[j], nums[i]
        left, right = i + 1, len(nums) - 1
        while left < right:
            nums[left], nums[right] = nums[right], nums[left]
            left += 1
            right -= 1
```

时间复杂度：`O(n)`

空间复杂度：`O(1)`

#### 287.寻找重复数

题目：给定一个包含 `n + 1` 个整数的数组 `nums` ，其数字都在 `[1, n]` 范围内（包括 `1` 和 `n`），可知至少存在一个重复的整数。假设 `nums` 只有 **一个重复的整数** ，返回 **这个重复的数** 。你设计的解决方案必须 **不修改** 数组 `nums` 且只用常量级 `O(1)` 的额外空间。

思路：把数组当成链表，找环的入口（入口值就是重复数）

```python
class Solution:
    def findDuplicate(self, nums: List[int]) -> int:
        slow = fast = 0
        while True:
            slow = nums[slow]
            fast = nums[nums[fast]]
            if slow == fast:
                break
        slow = 0
        while slow != fast:
            slow = nums[slow]
            fast = nums[fast]
        return slow
```

时间复杂度：`O(n)`

空间复杂度：`O(1)`
